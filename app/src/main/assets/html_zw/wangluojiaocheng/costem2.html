接下來我帶來的是手機端附魔教程哦！

1-準備好附魔用品

  1. 你的經驗等級(最好大於等於30級)
  2. 你要附魔的東西
  3. 必需的物品→青金石
  4. 書架
  5. 附魔台
  6. 鐵砧

2-擺好附魔台和書架

3-現在就可以附魔咯！

(我用鑽石劍來代表）

  1. 點附魔台把鑽石劍放進去。

  2. 是不是看見三條附魔欄是黑色的。那麼就開始放青金石了。

  3. 選擇你要附魔那一欄附魔就行了，那麼就可以了。當然要消耗經驗了～

4-現在是進階附魔

  1. 放置好鐵鑽

  2. 準備一本附魔書，我用的是耐久Ⅰ
  3. 放入鑽石劍和附魔書，消耗經驗完成附魔

  4. 那麼你又完成了。

5-現在帶上你的武器去征服這個地圖吧
