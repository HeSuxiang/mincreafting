第一章:學習Java

java是什麼？java是一門計算機語言，我相信很多人都知道這一事實，這很可能是由於您曾經玩過由這一語言編譯而成的遊戲。Java是一門面向對象的計算機語言，也就是說它由兩主要部分組成：字段和方法。那什麼是字段和方法呢？我們以人為例：

我們可以把一個人比作一個工程（o(∩_∩)o好像有些類比不當~）。在這個工程中會有許多包，比如「human.body.head」。在每個包中會包含好多類（我們之後詳細闡述），比如說"ear.java"就是一個類。現在我們已經知道了字段和方法，那麼什麼是類呢？我們以之前的ear.java為例，這個耳朵類可以諸如「耳垂高度」和「音量」等特徵，所以一個解決方案是建立如「listenTo()」和"moveEars()"等函數。

您可能會問這到底是什麼呀，也可能會問這些和Minecraft到底有神馬關係？好吧，以下就是一個例子：

方法/函數舉例 


public int getMaxHealth()
{
	return 20;
}

這是minecraft中EntityCreeper.java中的一段代碼。這段代碼非常基本，您只需關心的是如果把返回值20改為其它的什麼東西會發生什麼呢？當然您有可能會猜測：嗯，有可能會帶來我砍苦力怕多少刀它會死呢？其中的"public"表示其它的類也可以調用該函數，"int"表示該函數為整型函數（在+/-2,147,483,647之間的所有十進制整數）。"getMaxHealth()"為函數名，"return 20"表示返回值，比如如果有人問這個類：「苦力怕的最大生命值是多少呀？」，它將回答：「是20~」。由於我們在函數頭聲明瞭"int"，所以我們的返回值必須為整數。

字段/變量舉例


private double velocityY;

這是在EntityMinecart類中所生命的一個字段。其中"private"表示只有這個類才可以訪問這個字段，"double"表示為一個十進制數，"velocityY"為這個變量的變量名（字段和變量有時表示同一個東西）。
