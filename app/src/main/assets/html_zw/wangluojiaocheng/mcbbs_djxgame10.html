第四章第六節：創建一組工具

在接下來的教程中，我們主要集中於講述如何創建一組工具。

這一節要比其他各節都要複雜，所以各位同學打起精神來~~

首先，我們需要為新的工具創建一個EnumToolMaterial。

在您的類中添加如下代碼，注意不是在任何的類函數中。

New Tool Material

 
static EnumToolMaterial BLUETOOLS = EnumHelper.addToolMaterial("BLUE", 2, 171, 5F, 1, 5);

名稱含義

EnumHelper.addToolMaterial		我們所引用的Forge類函數。如果您在此處出現錯誤，請手動導入net.minecraft.forge.EnumHelper。
以下變量可以進行更改
''BLUE''材料名稱
2您所設定的工具能夠正常開採的硬度等級（如黑曜石礦為3、鑽石礦為2、鐵礦為1）
171該工具的耐久度（木質工具為59、鑽石工具為1561）
5F使用該工具開採的速度，您可以根據如下數據進行大概地判斷（木質工具：2.0F，鑽石工具為8.0F）
1該工具對實體所造成的傷害值，具體計算公式如下（該數值+4）÷2。（如此時每擊傷害為2.5顆心。）
5可附魔性，此處不確定（石頭為5、鑽石為10、金為22）

第二步，我們需要創建一個新的物品對象：

New Items

 
public static Item blueStoneShovel = new BlueStoneSpade(ItemID1, BLUETOOLS).setIconIndex(0).setItemName("customShovel");
public static Item blueStonePickaxe = new BlueStonePickaxe(ItemID2, BLUETOOLS).setIconIndex(1).setItemName("customPickaxe");
public static Item blueStoneAxe = new BlueStoneAxe(ItemID3, BLUETOOLS).setIconIndex(2).setItemName("customAxe");
public static Item blueStoneHoe = new BlueStoneHoe(ItemID5, BLUETOOLS).setIconIndex(4).setItemName("customHoe");
public static Item blueStoneSword = new BlueStoneSword(ItemID4, BLUETOOLS).setIconIndex(3).setItemName("customSword");

此時會報錯，這是由於還沒有新建類。

BlueStoneSpade我們接下來所要創建的類的名字
(ItemID1, BLUETOOLS)為您的鏟子所設置的物品ID。請記得為您每一個工具都設置一個不同的物品ID！BLUETOOLS是我們之前建立的EnumToolMaterial。
.setIconIndex(int)與之前類似，用於設定物品的子圖號
.setItemName(String)與之前類似，設定物品的內部名稱。每個物品都要不同！

現在創建一個新的類，此時您應該需要5個類了。現在我們執行下一步，擴展工具類（不是擴展item類）。

舉例來說，我的藍石鏟(BlueStoneSpade)會擴展為ItemSpade。那麼藍石斧(BlueStoneAxe)就應該擴展為 ItemAxe，以此類推。

Into load()

 
MinecraftForge.setToolClass(blueStonePickaxe, "pickaxe", 2);
MinecraftForge.setToolClass(blueStoneShovel, "shovel", 2);
MinecraftForge.setToolClass(blueStoneAxe, "axe", 2);


MinecraftForge.setToolClass		調用該類函數可以為我們的物品設置一個恰當的類
blueStonePickaxe我們傳遞的鎬的參數，也要記得為您的鏟子、斧頭進行相應的設置。
"pickaxe"小寫，工具類型
2之前提到的硬度等級，您之前應該通過EnumToolMaterial進行了硬度等級的設置

您只需要為斧、鎬、鏟進行相應的設置！

現在使用Modloader.addName添加名字，以及參照之前的教程添加合成表~
