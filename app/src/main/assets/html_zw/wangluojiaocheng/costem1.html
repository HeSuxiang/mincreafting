我的世界命令教程(國服手機端)

此教程由網友 水瓶 手打製作
非常感謝水瓶!

1-進入遊戲，(注意必須開啓作弊模式)如果你聯機，你不是房主的話，有些命令是用不了的！

2-點開對話框，一般都在遊戲界面的正上方中央。

3-如果你覺得背包東西很多的話，可以試下以下代碼。

clear 清除對象目標物品
格式:
/clear 目標 物品 數量

例子:/clear @a (清除所有玩家所有物品)

4-如果你想獲得物品可以試一下下面的命令。

give 給予玩家物品
格式:
/give 玩家 物品代碼 數量
例子:
/give @a diamond_sword 20 (給所有玩家20把鑽石劍)

5-如果你想殺人,又找不到人的話可以試下面代碼

kill 殺死某位玩家
格式:
/kill 玩家
例子:
/kill @a (殺死所有玩家，包括你自己)

6-如果你想召喚生物的話，你嘗試下面代碼

summon 召喚生物
格式:
/summon 生物代碼 數量
例子:
/summon mooshroom (召喚蘑菇牛)

7-接下來是常用代碼，看看你們有什麼需要的。

/time set day 設置時間為白天
/time set night 設置時間為夜晚
/gamerule dodaylightcycle false 停止時間流動
/difficulty 0 設置難度為和平
/difficulty 1 設置難度為簡單
/difficulty 2 設置難度為普通
/difficulty 3 設置難度為困難
/gamemode 0 設置模式為生存
/gamemode 1 設置模式為創造
/gamemode 2 設置模式為冒險
/weather clear 設置天氣為晴天
/weather rain 設置天氣為下雨
/weather thunder 設置天氣為雷雨
/gamerule keepinventory true 死亡不掉落(強烈推薦)
/spawnpoint 設置當前位置為出生點(強烈推薦)
/effect @p night_vision 99999 0 給予無限時長夜視(強烈推薦)
/tp ~ ~ ~ 傳送坐標(配合下面幾個命令效果極佳)
/locate stronghold 定位末影要塞
/locate endcity 定位末影城
/locate mansion 定位叢林府邸
/locate fortress 定位地獄堡壘
