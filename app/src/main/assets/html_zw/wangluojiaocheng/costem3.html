我的世界命名符彩蛋

此教程由網友 水瓶 手打製作
非常感謝水瓶!

又是我，這次我帶來的是我的世界彩蛋

彩蛋1-命名符(Dinnerbone/Grumm)

  1. 當然是要鐵鑽，命名牌和1級的經驗
  2. 放好鐵鑽，打開放入命名牌
  3. 把命名牌消耗一級經驗，更改為Dinnerbone/Grumm(第一個字母要大寫)
  4. 對準生物使用會讓生物上顛倒(玩家和末影龍無效)

彩蛋2-還是命名符(jed_)

  1. 還是鐵鑽，命名牌和1級的經驗
  2. 放好鐵鑽，打開放入命名牌
  3. 把命名牌消耗一級經驗，更改為jeb_
  4. 對準羊使用會讓羊變色(只對羊有用，還有用剪刀蒐集羊毛掉落的是沒有使用命名牌之前的顏色)

彩蛋3-你沒猜錯，還是命名牌

  1. 還是鐵鑽，命名牌和1級的經驗
  2. 放好鐵鑽，打開放入命名牌
  3. 把命名牌消耗一級經驗，更改為Toast(第一個字母要大寫)
  4. 對準兔子使用會讓兔子擁有特殊材質(ps:就是加了皮膚)
