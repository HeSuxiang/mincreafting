Minecraft MOD製作：從入門到精通——第四章（by djxgame）

一、基礎知識：JRE和JDK

1.JRE（Java Runtime Environment Java運行環境）

包括Java虛擬機（JVM Java Virtual Machine）和Java程序所需的核心類庫等，運行Java程序只需裝JRE即可。

2.JDK（Java Development Kit Java開發工具包）

Java開發工具，包括了JRE，其中的開發工具：編譯工具（javac.exe）、打包工具（jar.exe）等。

總的來說就是使用JDK開發java程序，JRE負責運行。

二、JDK下載與安裝

1.進入 http://www.oracle.com/technetwork/java/javase/downloads/index.html

2.找到Java SE（建議使用最新版），右邊點擊JDK下面的DOWNLOAD按鈕

3.點擊單選框Accept License Agreement

4.32位計算機選windows x86下載，64位計算機選x64下載

5.雙擊下載下來的文件進行安裝，其中安裝目錄可以更改，不建議安裝到C盤，因為重裝系統後就會沒了。另外其實也可以放到U盤里，也就是說JDK是可以移動的。

如果提示安裝Java sdk，可以取消，提示jre也可以不裝，因為jdk裡面包含了jre。

三、Java環境變量配置

我的電腦（計算機）-右鍵屬性-高級系統設置-環境變量。

1.在系統變量里點新建，變量名為JAVA_HOME，變量值為JDK的安裝目錄，比如我的是D:\Program Files\Java\jdk1.7.0_09

2.在系統變量里點新建，變量名為Classpath，變量值為.;%JAVA_HOME%\lib;（最前面有個點，最後面有分號，別漏了）

3.在系統變量里找到Path，點編輯，在變量值的最前端加上%JAVA_HOME%\bin;（注意有分號）

4.保存，開始菜單中「運行」，填「cmd」，確定。彈出黑框後輸入「javac」，如果有顯示java相關信息則表示配置成功。

5.如果有顯示java不是內部命令或外部命令則表示配置錯誤。請仔細檢查安裝路徑和環境變量.

四、配置工作環境

工具下載：MCP： http://mcp.ocean-labs.de/index.php/MCP_Releases

Minecraft Forge-scr： www.minecraftforge.net/forum/index.php/board,3.0.html

Eclipse sdk： http://www.eclipse.org/downloads/

本教程使用jdk1.7.0_09 、Eclipse sdk3.7.0漢化版 、minecraftforge-src-6.4.2.445 、mcp7.23

1.解壓mcp.zip到任意目錄，比如mcp，建議新建一個mod製作相關的文件夾。

forge安裝路徑不可以有中文

2.解壓forge-src.zip，將裡面的forge文件夾放進mcp文件夾里。

3.運行mcp\forge\install.cmd，直到運行完畢（按任意鍵結束）。

其中第3步會自行下載一些文件，如果下載太慢。可以自行下載下面的文件放進對應目錄，再運行install.cmd

Lib.zip-解壓並將裡面的所有文件放進mcp\lib內

jars.zip-解壓並將裡面的bin文件夾放進mcp\jars內
