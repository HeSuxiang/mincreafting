第五節:添加燒煉與創建物品

在這一節，我們將要為您在上一節創建的礦藏添加燒煉物，以錠為例，因為其相對較為簡單。

創建物品與創建方塊相比非常類似，事實上我們只需要在方塊/物品的代碼中修改一點點的地方即可。我們這次需要創建一個物品對象。

New Item Object

 
public static Item yourFirstItem = new ItemTutorial(280).setIconIndex(0).setItemName("itemName");

與方塊相比，除去使用的對象和類不同外其餘完全相同。

我們此時並沒有使用在結構體內添加子圖的方法，而是使用.setlconindex(int)類函數，.setitemName與其相同，

即使這些並沒有在遊戲中出現，但也應該與您的方塊不同。

現在我們創建ItemTutorial類，代碼與前文所述BlockTutorial類的代碼完全相同（您可以自行更改類的名稱）。

此時只需要擴展item類，注意並不需要覆蓋addCreativeItems類函數。

使用Modloader.addName()類函數、預加載子圖（如果您方塊和物品使用不同子圖重新預加載的話需要）

在load()函數中添加如下代碼：

Into load()

 
ModLoader.addSmelting(yourFirstOre.blockID, new ItemStack(yourFirstItem));

其中的第一個參數是輸入端，亦即我們之前製作的那個方塊，第二個參數為itemStack亦即我們通過熔爐燒煉出的您創建的物品。
