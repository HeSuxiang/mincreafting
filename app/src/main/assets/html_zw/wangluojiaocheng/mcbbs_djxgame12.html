第四章第八節：添加多材質方塊

什麼是多材質方塊？就是類似草地這樣子的方塊：綠色的頂部，低端是棕色的，而四周卻與上下均不相同。

第一步需要為其創建材質。需要繪制至少兩幅材質才可以實現多材質的功能，將其放在同一份子圖文件下。

確保使用渲染軟件將不同的材質渲染在同一方塊上使其看起來彼此拼接，畢竟您不希望一個帶有紫色邊的方塊轉過去一看編程綠色邊了吧……

在您製作成功材質後實際上代碼非常簡單，您只需要一個類函數即可：

Magical Multi-texture Method

 
@Override
public int getBlockTextureFromSide(int side)
{
    if (side == 1)
    {
        return 5;
    }
    else if (side == 0)
    {
        return 6;
    }
    else
    {
        return 4;
    }
}

在這一例程中展示了一份草方塊的製作過程。通過使用getBlockTexture()類函數，我們可以調用不同位置的子圖。

當您看到這一方塊時：

頂部：0

底部：1

四周：2、3、4、5

請確認您所看到的方塊是否與其吻合~~（據反饋此處有問題……）
