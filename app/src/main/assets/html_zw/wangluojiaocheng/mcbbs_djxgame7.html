第三節：創建一份合成表

在這一節，我將為您講述如何創建一份合成表，本章不會有任何下載內容O(∩_∩)O~

當您已經成功地掌握了前兩節的內容，您就已經基本掌握了製作MOD的基本知識。

現在需要為那些奮鬥在生存模式的人們考慮了~

本節內容十分簡單，僅調用一個類函數。

所以您可以嘗試一下能否自己一個人搞清楚代碼，我會將釋義附在後面。

當然，這些代碼完全可以一行寫完，但是寫成這樣子更為直觀，就好像您在用工作台進行合成一樣。 

 
ModLoader.addRecipe(new ItemStack(yourFirstBlock), new Object[] {
    "BBB",
    "BAB",
    "BBB",
    Character.valueOf('A'), Item.diamond,
    Character.valueOf('B'), Block.obsidian
});

您看懂了麼？

名稱 含義
ModLoader.addRecipe 這個是您所調用的類函數，用於為Minecraft添加合成表。
new ItemStack(yourFirstBlock) 新建一個ItemStack類函數，該類函數的作用是允許工作台合成該物品或方塊。您可以將 "yourFirstBlock" 改為您之前創建的物品或方塊
new Object[] 這個無需關心，在大括號內添加的信息將會以對象的形式傳入Modloader類函數中
"BBB", "BAB", "BBB", 此處為工作台中將會顯示的合成方式，請牢記為每一種不同的原料設置一個不同的字母。那麼為什麼我使用了兩個字母呢？是因為我只需要兩種不同的原材料……
Character.valueOf('A') 在上文中的「A」代表了如下這種物品……
Item.diamond, 咳咳，再合成公式中的每一個「A」均代表一塊鑽石
為您每一種原材料均設置 Character.valueOf('letterHere') 和Item.whatever
}); 結束

您當然也可以在合成中使用您自己創建的方塊或物品，

舉例來說，"Character.valueOf('A'), yourFirstBlock"即代表之前所述的「A」為您所創建的方塊。
