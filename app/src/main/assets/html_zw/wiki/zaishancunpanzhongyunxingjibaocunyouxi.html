在閃存盤中運行及保存Minecraft

此教程解釋了如何在閃存盤上運行Minecraft。你在使用不同的電腦上使用相同的遊戲數據(世界、保存的數據、資源包、服務器等)時這很有用，因為遊戲數據將會始終保存在便攜式驅動器中。

先決條件

你需要：

  • 一個最少擁有 512MB 內存的便攜式驅動器。你將會需要更多的空間在驅動器上儲存你的Minecraft存檔。
  • 一個運行著Windows、OS X、或任何Linux的發行版的電腦，這台電腦要符合Minecraft 配置運行需求。

步驟

安裝了Windows的電腦

準備工作

因為 Minecraft 需要安裝多個文件，我們將會收集 Minecraft 運行所有的所需文件，並在便攜式驅動器中分好類，以方便尋找。請注意，如果你不正確的按照下面的指示做，那麼 Minecraft 可能會崩潰或啓動失敗。Windows 對所有的文件和文件夾名稱的大小寫都很敏感。

  • 將便攜式驅動器插入到你的電腦中，並打開它。
  • 新建一個文件夾，將其命名為 Minecraft Portable(或其他你選擇的名字，名字不能含有中文字符)，並進入文件夾。
  • 在這個文件夾里，新建2個文件夾，並分別命名為 Launcher 和 Data。
  • 注：這些文件夾必須有確切的名稱，因為我們將在以下步驟中使用它們。
  • 現在在驅動器上有3個文件夾：在 Minecraft Portable 裡面的 Launcher 和 Data。

啓動

往便攜式驅動器中添加啓動器/客戶端：

  • 下載 Minecraft launcher。
    • 你可以在https://minecraft.net/download中下載它。
    • 確保你使用了「Minecraft for Windows (alternative)」 版本，這可以在「Show all platforms」頁面中找到。
  • 將 Minecraft.exe 複製到已經創建的 Launcher文件夾。

數據文件

此部分教程將會指導你如何將遊戲數據(世界、儲存的數據、資源包、服務器等)複製到便攜式驅動器中。
如果你不想複製你的遊戲數據，你可以跳過這個部分。

所有的遊戲數據都必須在 .minecraft 文件夾里。

  • 按下 ⊞ Windows + R
    • 輸入 %APPDATA% 並點擊確定。
  • 查找 .minecraft 文件夾，並複製它。
  • 將複製的文件夾粘貼到 Data 文件夾。

設置

現在所有的數據文件都在這裡了，我們需要告訴 Windows 將 Minecraft 引導到在便攜式驅動器的文件上。為了達成這個目標，我們需要創建一個批處理文件(.bat)。

  • 打開記事本或一個簡單的文字編輯器。
    • 按下 ⊞ Windows + R。
    • 鍵入 notepad 並點擊確定。
  • 在文字編輯器中複製並粘貼下列文字：
 set APPDATA=%CD%\Data
 start %cd%\Launcher\Minecraft.exe
  • 如果你願意使用 minecraft.jar，使用下列文字代替：
 set APPDATA=%CD%\Data
 start %cd%\Launcher\Minecraft.jar
  • 現在點擊 文件 > 另存為...
  • 導航到你的 Minecraft Portable 文件夾(不是Launcher 或 Data)。
  • 命名這個文件為 "StartMinecraft.bat"。
  • 雙擊運行它。如果要修改這個文件，請右擊這個文件，並點擊「編輯」。

開始遊戲

  • 一旦設置完成後，你就可以通過運行已經創建的 StartMinecraft.bat 文件來啓動 Minecraft 了。
  • 當啓動器出現時，點擊 Play 開始遊戲。

安裝了 Linux 的電腦

我們要確定便攜式驅動器在 /dev/sdb1 中，並且 Minecraft 被安裝在 ~/games/.minecraft

  • 注：目錄名稱可能不同，這取決於你將 Minecraft 安裝到的目錄。

在一個 Terminal 中運行下列命令(確保以管理員方式運行，或者使用 sudo)：

 umount /dev/sdb1
 mkdir /media/minecraft
 mount /dev/sdb1 /media/minecraft
 mkdir /media/minecraft/games # 如果你的 USB 中有一個儲存遊戲的文件夾，那麼可以跳過該步驟。
 cp - rn ~/games/.minecraft /media/minecraft/ # 這可能需要花費一些時間。
 cd
 umount /dev/sdb1
 rm /media/minecraft # 雙擊以確保你的 USB 已經安裝！

在此之後，如果你想遊玩 Minecraft，只需要在 Terminal 中輸入下列命令。(你還是需要超級用戶權限)

 umount /dev/sdb1
 mkdir ~/games # 如果遊戲文件夾在根目錄里，那麼可以跳過。
 mount /dev/sdb1 ~/games # 你將不能夠玩任何儲存在 ~/games 目錄里的遊戲。

運行你的 Minecraft 啓動器並開始遊玩。

安裝了 OS X 的電腦

這部分並不是真的是讓 Minecraft 在便攜驅動器上運行，因為它需要在你每次進行遊戲時複製遊戲數據到便攜驅動器上面。

設置

  • 尋找你的 Minecraft 文件。當在搜索界面時按下 Command+Shift+G ，然後複製 ~/Library/application support/minecraft 到對話框中。
  • 將在你的 application support 文件夾里的 minecraft 文件夾複製。
  • Insert in your portable drive and make 2 folders in it: 插入你的便攜驅動器，並新建2個文件夾： data 和 launcher
  • 將 minecraft 文件夾粘貼到便攜式驅動器的 「data」 文件夾中。
  • 拖動你的啓動器到「launcher」文件夾。
  • 移除你的便攜式驅動器。

開始遊戲

一旦完成設置後，你就可以在另一台 Mac 上測試你的 Minecraft 是否可以運行了。

  • 將便攜式驅動器插入到另一台要測試的 Mac。
  • 打開 launcher 文件夾，並移動 Minecraft launcher 到你的 Applications 文件夾。
  • 雙擊啓動器來運行它。在 Minecraft 加載後退出。
  • 轉到 Users/USERNAME/Library/Application Support
  • 在 「Application Support」中，刪除 minecraft文件夾。
  • 將便攜式驅動器里的 data 文件夾裡面的 minecraft 文件夾移動到桌面。
  • 將該文件夾從桌面移動到 application support。
  • 再次運行 Minecraft。

補充注釋

注：如果你往便攜式驅動器里安裝資源包或 Mod，那麼當有響應的指示讓你複製文件時，你就需要使用到你的閃存盤裡面的 .minecraft 文件夾。

注：該方法同樣適用於 Dropbox、Google Drive 和 OneDrive。

Cryshal 的方法

不工作，minecraft 始終還是從 appdata 文件夾中啓動，Minecraft Profile Manager 沒用，Java 也不能從閃存盤中運行

你需要

  • 1個閃存盤(建議容量在2GB左右，但是這取決於你擁有多少世界/資源包)
  • Windows 系統


  • 首先，創建一個文件夾，例如「Flash MC」。然後，下載 jPortable，這個軟件可以讓 Java 在你的閃存盤上正確運行，而不用安裝在電腦後才能運行。
  • 你需要將 jPortable 安裝到閃存盤中的一個文件夾，如 \Flash MC\jPortable。
  • 然後，前往 Minecraft Profile Manager的帖子，並下載它。
  • 按照說明將它放入到你的閃存盤裡面的 MC 文件夾。(將 minecraft.exe 放置在相同的文件夾)
  • 現在你將能夠啓動 MPM，新建一個用戶資料，並下載運行 Minecraft 的全部所需文件。

注意

  • MPM的運行需要使用 Java，這意味著你必須安裝 Java 來讓它工作。
  • 運行 Windows 的電腦應該很好的工作，除非它們沒有打補丁。

希望這能夠幫到你！

HiWiki's 的方法

警告
請小心跟隨下面的步驟，以防止 Minecraft 崩潰！
該方法來源於 HiWiki！

警告 - 請獨自承擔這帶來的風險！
只在 WINDOWS 上工作！

你需要：

  • USB (最低要求容量為2.5GB，如果你想安裝 Mod 或材質包，你就需要更多空間(你也需要額外的空間存儲世界))
  • 記事本
  • Minecraft 客戶端
  • .minecraft 文件夾

步驟

  • 首先，你需要新建2個文件夾——bin 和 data。
  • 然後，將 Minecraft 客戶端放入 bin 文件夾。
    • 如果你使用 jar 文件(或其他運行 Minecraft 的東西)，也請放置到 bin文件夾。
  • 但是 Minecraft 的數據並不在 data文件夾中，
    • 因此你需要按下 ⊞ Windows + R ，打開運行窗口，並在窗口中輸入 %appdata% ，然後複製 .minecraft 文件夾(同時包括 minecraft 的其它文件夾，例如 .aether)
      • 選擇這些文件夾並按下 Ctrl + C (或 Ctrl + X 如果你想移動這些文件)。
  • 然後再次按下 ⊞ Windows + R ，並在對話框中輸入 notepad，然後複製下列代碼：
 SET APPDATA=%CD%\data
 bin\minecraft.exe
  • 如果使用 jar 文件，則用下列命令：
 SET APPDATA=%CD%\data
 bin\minecraft.jar
  • 如果使用的 Jar/Exe 文件擁有不同的名稱，例如 Minecraft-launcher，則用下列命令
 SET APPDATA=%CD%\data
 bin\<啓動器的名稱>.<jar/exe>
  • 這就完成了！

注意！

  • 該方法將會需要 Java，如果你想讓 java 在你的閃存盤中運行，請查看下面的部分——添加便攜式 Java 的方法
  • 在 appdata 的 .minecraft 文件夾裡面的數據都要在你的閃存盤中運行。

添加便攜式 Java 的方法

該方法需要額外的 0.5GB 磁盤空間！

  • 現在你已經完成上面的步驟！
  • 是時候添加便攜式 Java 了(YAY!)
  • 首先，點擊 這個鏈接。
  • 然後，在頁面上點擊Download Now或這個鏈接。
  • 然後打開 PortableApps.com_Platform_Setup_11.2.exe ，在你看見 add apps section 前請等待。
    • 確保你已經將安裝程序放置在閃存盤中！
  • 然後在工具里尋找並勾選 jPortable 、jPortable Launcher、jPortable Browser 的復選框，然後點擊 install。
  • 等待完成後，就完成了！

如何開啓 jPortable

  • 首先，在你的閃存盤中尋找文件夾 PortableApps。
  • 然後雙擊 start，並點擊 jPortable Launcher。
  • 這將會打開一個窗口，然後尋找 jar。(如果只顯示 jar，不用擔心，只要將它改為 all files，然後尋找你的 exe 即可)
  • 之後點擊 open，然後你的 Minecraft 將會打開。

漏洞與修復

  • 如果找不到 minecraft 目錄，請確認是否點擊了 edit profile 按鈕，並在資源管理器中定位到你的 data 文件夾，然後點擊它，並按下 properties。
  • 同時也可以編輯 edit profile 窗口裡面的 Game directory 文本框。
  • 現在應該出現了一個窗口，你將要對拖動文本的位置的一行進行選擇，然後將其粘貼到 Game directory 文本框中，並輸入 \.minecraft 或複製它到 Game directory 文本框中。
  • 然後你就完成了！

通過修改autorun.inf以修改啓動器文件

如果你autorun.inf內容是這樣的：

 set APPDATA=%CD%\data
 bin\minecraft.exe

或：

 set APPDATA=%CD%\data
 bin\minecraft.jar

請將文件內容改成這樣：

 set APPDATA=Minecraft portable\data
 cd Minecraft portable\drive\
 Minecraft.jar
