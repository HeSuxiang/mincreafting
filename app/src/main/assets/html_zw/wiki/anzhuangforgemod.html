安裝 Forge Mod

該頁面的內容不受Mojang AB、Minecraft Wiki、Minecraft IRC頻道與Minecraft論壇的官方支持。

Minecraft 1.6 及之後的版本

跨平台安裝

  1. 轉到 files.minecraftforge.net，如果使用下面的鏈接，跳到第2步。
  2. 找到你的 Minecraft(必須是 1.6+)對應最新或推薦的版本。下載鏈接在 Minecraft 版本的旁邊，鏈接名叫「Installer」。
  3. 下載後，運行它(Windows)或在列表或文件夾中雙擊它(OS X 和 Windows)。Linux 用戶需要手動運行 JAR 文件，使用如 java -jar <installerName> 這樣的命令。
  4. 這將會打開一個叫做「Mod System Installer」的窗口。點擊「Install Client」復選框。Th
  5. (可選)如果你想改變安裝目錄，點擊顯示著默認安裝目錄路徑旁邊的「...」按鈕(.minecraft 文件夾)，然後選擇一個你想安裝的文件夾。
  6. 點擊 Install 並等待安裝完成。
  7. 打開新的 MinecraftLauncher ，選擇「Forge」Profile，或點擊「Edit Profile」並改變版本為「release x.x.x-Forgex.xx.x.xxx」(「x」為構建編號，例如1.6.4-Forge9.11.0.884)。

安裝 Minecraft Modinstaller

Minecraft Modinstaller 能快速、容易地安裝 Mod。你可以在開發者網站http://www.minecraft-installer.com 下載它。工具推薦 Mod 會根據你選擇的 Minecraft 版本安裝的 Mod 進行推送。全高清圖像和評論描述會幫助你進行選擇。目前，已超過150個 Mod可供下載。

  1. 在列表裡選擇你的 Minecraft 版本。
  2. 在左側的列表尋找你喜歡的 Mod。
  3. 點擊「Add」箭頭來在右邊的列表進行選擇要安裝的項目。
  4. 點擊「Install」(所有的 Mod 都會下載，包括 Minecraft Forge)，然後 Minecraft 將會啓動。

Minecraft 1.5 及之前

這是舊版本 Minecraft 的 Forge API 安裝教程。現在1.6之後允許使用「簡單安裝器」安裝 Minecraft Forge。

Windows 安裝

  1. 選擇你想要的 Mod 並下載它們。通常，它們會說明 Mod 對應的版本——例如：mod - 1.7.2 jar。下載下來的文件將會是一個 Zip 或 Java 文件。
  2. 現在你需要下載 Minecraft Forge，鏈接在最下方。你需要下載 Mod 對應版本的 Forge。如果你下載了 1.7.2 Forge，你需要所有的 1.7.2 Mod。
  3. 如果你尚未啓動過 Minecraft，運行它。
  4. 允許 Forge Client installer。
  5. 定位到你的 .minecraft folder。你可以點擊開始菜單，然後在搜索框中輸入%appdata%，點擊 roaming，然後點擊 .minecraft。XP 用戶要使用運行來代替搜索。
  6. 定位到你最近創建的「mods」文件夾。
  7. 把你下載的 Mod 拖到文件夾里。
  8. 關閉文件夾，打開名叫 1.*.*-Forge**.**.*.**** 的新 Minecraft 版本(* 為版本號)。

Mac 安裝

  1. 不要忘記 Force Update！隨後關閉它。
  2. 通過 Finder 程序進入到你的 Library，並進入 Application Support 文件夾。
  3. 之後你會找到文件夾「minecraft」，轉到 bin 文件夾，並用 The Unarchiver 程序解壓 minecraft.jar 的文件。
  4. 安裝 Forge 版，它應該會自解壓。它包含幾個 classes。
  5. 把新 classes 逐個 放置到解壓文件夾中。你也可以刪除 META-INF，如果你希望(推薦這麼做)。不要忘記移除 ibxm 文件夾里的材質！
  6. 測試 Minecraft，如果有菜單彈出，你就成功安裝 Forge 了！
  7. 在 finder 里添加你想添加的 Mod 到 mods 文件夾中。(或者 coremods 文件夾，如果它是一個 coremod)，然後盡情享受吧！
