在服務器上遊玩

Minecraft 是一個單人遊戲(除非你在多人遊戲上遊玩)。你獨自生存，並且你的朋友只有動物，而且你可能會把它們殺了來獲得食物！你可能想和朋友一起漫步世界。只需和其他玩家和可以分享爬行者的更多生物一起體驗。因此服務器將要在 Minecraft 世界的方塊里做標記。這裡是進一步的說明。

加入服務器

  • 在 Minecraft 服務器列表網站上尋找一個服務器。
  • 閱讀服務器介紹並選擇一個你喜歡的服務器。找到它的IP地址並把它複製到剪貼板(Windows系統使用:Ctrl+C；Mac系統使用:Cmd+C)。IP地址通常是一串數字，通常寫在服務器名稱的後面，或者是一個域名的後面。
  • 啓動Minecraft，點擊「多人遊戲」後再點擊添加服務器。 這裡應有兩個文字框,分別是服務器名稱和服務器IP地址。 服務器名稱並不重要，你可以隨意輸入。現在按下Ctrl+V(在Windows操作系統中)將服務器IP地址複製到框中。然後點擊完成來保存服務器。
  • 此時你應返回到遊戲中的服務器列表。服務器應該會在上面出現，右邊會有綠色的條。
  • 點擊這個服務器,再點擊加入。正常來說，你將會被帶到這個服務器。如果沒有，下面列出了一些常見的錯誤來幫助你解決這個問題。

可能出現的錯誤

有時候事情會出錯，因此這裡有一些錯誤信息和他們代表的含義。點擊加入服務器時可能會出現如下信息：

被服務器中斷連接 或 過期的服務器！

如果Minecraft最近更新了，你可能會接收到這些錯誤信息。它們的意思是你運行的客戶端版本與服務端的版本不同。你可以在啓動器創建新的與服務端同版本的profile，然後等待服務端更新，服務端更新的時間取決於服主的反應。

你不在這個服務器的白名單列表上!

一些服務器開啓了白名單系統來防止其他人加入服務器——通常的，如果要被加入到白名單，你不得不訪問一些外部網站，通常是服務器的網站。

您的賬號已被此服務器封禁！

您的用戶名或IP已經被服務器封禁。通常意味著你做了一些違反服務器規則的事情。要解除封禁，你需要聯繫服務器的管理員以請求解封。如果一個你從來沒有聽說過的服務器封禁了你的 IP，意味著你的IP地址被封鎖了。

被服務器中斷連接。無效的會話(請嘗試重啓你的遊戲)

當你登錄到 Minecraft 時，你會獲得一個 Mojang 服務器追蹤會話 ID。你的賬號只有最新的會話 ID 才被允許加入服務器(即使離線模式關閉)。按照這個消息去做，重啓你的遊戲。這可能是其他人登錄過你的賬號，為了防止這樣的事發生，更改你的密碼。你不能使用假冒的或破解的 Minecraft 連接服務器。

無法連接到服務器

該消息出現在服務器列表的旁邊。這意味著你無法連接到這個服務器。這可能是因為服務器已經關閉，你沒有網絡連接，或者服務器響應超時(因為延遲)。

Cannot resolve hostname

該消息出現在服務器列表的旁邊。這意味著找不到指定名稱的服務器，檢查地址並重試。

數據流終止

如果你試圖以過快的速度連接服務器，就可能會出現這條消息；斷開連接，等一會兒並嘗試重新連接，這會有效的。

身份驗證目前正在停機維護。

Minecraft 服務器檢查會話 ID 的功能當前不可用。你可以在 help.mojang.com 的 Service Statuses 處查看服務器的當前狀態

Java 錯誤，例如 java.lang.aMethodWhichReturnedAStackTrace(): 一個錯誤消息

該錯誤的發生原因取決於該錯誤的消息。在網絡上搜索這個問題，你可能會找到一個解決方案。

緩衝區溢出

查看維基百科上的緩衝區溢出詞條以獲取更多信息。

在服務器上發生的事情

在 Minecraft 服務器上會存在多個原型，可以通過獨特的遊戲特性、規則和引入的結構來區分。2個服務器不會是相同的，並且原型之間的界線經常是模糊的或無法區分的。許多特殊類型的服務器會依靠服務端軟件進行修改，例如 Bukkit 。服務端軟件並不受 Mojang 的支持。Bukkit(以及其他服務端軟件)服務器可能會帶有修改原版服務器體驗的插件。如果是 Bukkit 服務器，可以輸入 /plugins。然後你就能搜索插件的用途了。

要查看常見的服務器類型，詳見服務器頁面。

對於未更改的服務端軟件來說，這些是真的：

  • 你將會在服務器的生成點處生成。生成點默認有 16 保護區，在這片區域里你不能破壞方塊或與物品進行互動。
  • 在屏幕的左下側是聊天區域，人們會在這裡發送信息。默認聊天按鍵為 T 鍵。你也可以向聊天框里輸入命令，這與聊天內容是完全不同的事情。要查看可用的命令列表，輸入 /help
