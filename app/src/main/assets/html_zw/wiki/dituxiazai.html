地圖下載

在玩家的遊戲中生成的地圖可以被其他玩家下載並進行遊玩。

地圖存檔的存儲位置取決於其所使用的操作系統。目前選擇 .minecraft 數據文件夾是不可能的，因此世界存儲在一個已建立的文件夾里。

在 Beta 1.3 之前有5個地圖存檔位置，並且不能自定義它們的名稱。為了使用5個以上的世界，就要重命名或移動文件夾。這不再必要了，因為理論上可以擁有不同數量的世界，每個世界都可以擁有它們自己的名稱。

Windows

先決條件

一個壓縮軟件，例如 WinRAR 或 7-Zip。一些文件如 .RAR 不能在沒有壓縮軟件的情況下進行解壓，而在沒有壓縮軟件的情況下可以處理 .ZIP 文件。

獲得世界存檔

使用網絡瀏覽器(如 IE瀏覽器、Chrome 瀏覽器或火狐瀏覽器)下載存檔。

解壓文件

如果你下載下來的文件是 .ZIP 或 .RAR 文件的話，那麼就有必要進行這個步驟。

  • 解壓文件。右鍵，然後選擇解壓。如果文件格式是 .RAR，那麼 Windows 將不能夠打開它，這是你就需要一個壓縮軟件如 WinRAR 或 7-Zip。
  • 如果你看見一個叫做data的文件夾和一個叫做session.lock的文件，你就可以繼續進行下一步了！將文件夾解壓到你的桌面。
  • 如果看不見，那麼文件可能存在，不過你得挖得更深才能看見。一些存檔可能擁有2個文件夾，叫做Mac OS X 和 your_save。檢查每個文件夾，直到你看見data文件夾和session.lock文件。一旦找到了，就將文件夾解壓到你的桌面。

將文件放入 Minecraft 存檔存儲位置中

現在我們擁有了文件，我們將要把它們放到 Minecraft 的存檔文件夾里。

  • 複製在桌面上的文件夾(複製data的上一級目錄和session.lock，例如，如果data文件夾和session.lock存儲在 your_save_multi.RAR 的Mac OS X的your_save中，那麼就複製your_save文件夾。)
  • 按下 ⊞ Windows + R 打開運行窗口，並輸入 %appdata%\roaming\.minecraft\saves ，然後點擊 ↵ Enter，你將會進入 Minecraft 的 saves 文件夾，然後你就會留意到你的世界也在這裡。
  • 在saves文件夾中按下 CTRL + V 以粘貼文件夾。
  • 檢查your_save文件夾，確保文件夾里存在下面的文件/文件夾。
    • data
    • DIM-1
    • DIM1
    • players
    • region
    • level.dat
    • levl.dat_mcr
    • level.dat_old
    • session.lock
  • 一旦確定它們都在這裡了，關閉 .minecraft 文件夾，並啓動遊戲。

在 Minecraft 中尋找存檔

這裡是最難的部分：在你的儲存的所有存檔中尋找你的新存檔。

  • 啓動後，點擊「單人遊戲」。
  • 你複製的文件夾明將會顯示在世界名稱的下面。大多數存檔會叫做「world」。
  • 新存檔通常都在最下面，但是也有例外，你將會需要尋找存檔。

找到存檔後，加載它並開始遊玩吧！

Mac

尋找世界存檔

從瀏覽器(如 Safari、Chrome 瀏覽器或火狐瀏覽器)中下載存檔。

解壓文件

(有時候下載的文件是沒有被壓縮的。檢查文件是否為 .zip 或 .rar。)

  • 雙擊(或右擊並選擇打開)以解壓文件夾。
  • 你將會看到在解壓的文件夾里有一個文件夾。

將文件放置到 Minecraft 存檔位置中

將文件夾移動到 /Users/[USER]/Library/Application Support/minecraft/saves。
如果你的 minecraft 文件夾在不同位置，使用那個不同的位置代替。

解決

如果在 Minecraft 中找不到對應的存檔，請：

  • 檢查你是否將一個文件夾放到 saves 文件夾里。
  • 檢查你是否忘記解壓文件夾。
  • 檢查 Minecraft 是否使用了不同的 minecraft 文件夾。

Linux

該教程基於 Ubuntu 14.04.1 LTS 編寫。其他的發行版本可能使用不同的資源瀏覽器，但是 .minecraft 的位置都是相同的。

先決條件

一個壓縮軟件，如 Archive Manager (文檔管理器)。一些文件如 .RAR ，將在不能夠在沒有解壓的情況下打開。ZIP 文件可以被 Archive Manager 打開。
