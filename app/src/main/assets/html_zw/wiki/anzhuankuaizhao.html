安裝快照

安裝快照可以瞭解到在接下來的 Minecraft 版本中的新內容，然而許多人不知道怎麼安裝快照，因此開設本教程。根據本教程，你將學會如何安裝快照。

1.6 以後

為安裝並開始一個1.6的快照版本，先下載啓動器。這是可用的下載鏈接minecraft.net/download

默認的，你可以選擇 Minecraft 的版本並使用該版本進行遊戲(如1.5.2，1.5.4和1.7.2)。為了安裝開發版本，請按照以下操作：

  • 為開發版本創建新的用戶資料(Profile)。點擊啓動器上的 New Profile 按鈕，將會彈出一個對話框。
  • 命名這個版本(如 'Snapshots')。為防止不必要的麻煩，請盡量不要使用中文對該版本進行命名。
  • 點擊 Game Directory 復選框，並在文本框後面添加一個文件夾路徑(如 "C:\")。
  • 點擊 Enable experimental development versions(snapshots) 復選框。
  • 選擇你喜歡的快照版本
  • 點擊 Save Profile。

現在，你可以在啓動器上使用最晚的版本和開發版本進行遊戲。注意 Minecraft 不會檢查數據的兼容性。當你進行開發時，你應該使用一個臨時的世界進行開發。(請注意：有可能快照沒有改變遊戲目錄，然而將地圖從遊戲目錄中分離可以防止數據被破壞。)

如果你想比較舊的快照，點擊啓動器上的 Edit Profile 並在 Use versions 下拉選擇框中選擇一個版本。

1.6之前

為了安裝 1.5 和較早的快照，請按照下列步驟：

第一步： 轉到 快照歷史 頁面，選擇快照版本並在右面的「下載鏈接」中下載快照。
第二步： 轉到 .minecraft 文件夾。

  • Windows XP/Vista/7/8: %appdata%\.minecraft\(只需將這個地址複製到你的地址欄並點擊回車)
  • Linux: ~/.minecraft/ (該文件夾可能在 「Home」 文件夾中隱藏)
  • Mac OS X: ~/Library/Application Support/minecraft/ (該文件夾可能被隱藏)

第三步： 使用你下載的快照替換 "minecraft.jar" 的 "bin" 文件夾，然後啓動 Minecraft。

  • Linux 用戶，如果想取消隱藏文件夾，請按 Ctrl+H 。

恭喜，你已經成功安裝快照！

警告1: 打開已存在的單人世界存檔並不是沒有風險：該存檔可能會被破壞或無法加載。如果你在新快照中加載地圖並且這個新快照比當前版本要低，為了避免風險，請備份存檔！(存檔在 \.minecraft\saves\)

警告2: 如果你為你的 Mod 存檔進行備份，那麼你應該為 minecraft.jar 進行備份。只需將這個文件複製到一個安全的位置。(這個文件在 \.minecraft\bin)


如果想恢復到原來的版本，請在啓動器的選項中點擊 "Force Update"。
