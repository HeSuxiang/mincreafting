加載資源包

這些教程幫助你在 Minecraft 中使用資源包。資源包允許玩家自定義材質、音樂、音效、語言文件、致謝、閃爍標語以及字體，但不會改變實際的遊戲內容。

全平台

  1. 下載一個資源包，資源包應該是一個 .zip 文件
  2. 打開儲存下載文件的文件夾
  3. 複製 .zip 文件
  4. 打開 Minecraft
  5. 點擊「選項」，然後點擊「資源包」
  6. 選擇「打開資源包文件夾」"
  7. 向 Minecraft 資源包文件夾粘貼文件
  8. 回到 Minecraft，資源包應該會顯示在列表中，選擇它並點擊「完成」就能更換資源包了。

直接方法

Windows

  1. 下載一個資源包，下載的文件應該是 .zip 文件
  2. 打開存儲下載文件的目錄
  3. 複製 .zip 文件
  4. 按下 ⊞ Windows + R ，然後輸入 %appdata%/.minecraft/resourcepacks
  5. 向文件夾粘貼文件
  6. 打開 Minecraft
  7. 前往「選項」，然後點擊「資源包」
  8. 資源包現在會被列出來了，你可以選擇它並點擊「完成」來激活材質包

Mac OS X

  1. 下載你選擇的資源包
  2. 請勿將資源包文件解壓，而是將它移動到 Minecraft 的資源包文件夾(應該在 ~/Library/Application Support/minecraft/resourcepacks)
  3. 打開 Minecraft 並登錄
  4. 在主菜單選擇「選項」，然後選擇「資源包」
  5. 現在選擇你的新資源包並點擊完成
  6. 資源包現在會被列出來了，你可以選擇它並點擊「完成」來激活材質包

要在 Mac OS X 10.7 及更高版本系統訪問 ~/Library/ ，轉到 Finder，打開「Go」菜單，在按住 ⌥ Option 時選擇「Library」

Linux

  1. 下載一個資源包，下載的文件應該是 .zip 文件
  2. 打開存儲下載文件的目錄
  3. 複製 .zip 文件
  4. 在你的控制台文件管理器地址欄輸入 ~/.minecraft/resourcepacks/
  5. 向目錄粘貼文件
  6. 打開 Minecraft
  7. 轉到「選項」，然後轉到「資源包」
  8. 資源包現在會被列出來了，你可以選擇它並點擊「完成」來激活材質包
