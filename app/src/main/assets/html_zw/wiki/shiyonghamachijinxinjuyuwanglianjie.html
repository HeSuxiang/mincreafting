使用 Hamachi 進行局域網連接

該頁面的內容不受Mojang AB、Minecraft Wiki、Minecraft IRC頻道與Minecraft論壇的官方支持。

Hamachi 局域網

你可以設置一個 Hamachi 局域網，讓處於 Hamachi 網絡里的人通過 Hamachi 連接到你的局域網。

  1. 安裝Hamachi。你要安裝的是非托管版。
    https://secure.logmein.com/products/hamachi/download.aspx
  2. 在 Hamachi 創造一個 VPN 隧道。
  3. 讓你的朋友進入你的 Hamachi 網絡。
  4. 讓服務器里的每個人把 Hamachi 網絡設為家庭網絡，或者允許 Minecraft 使用公共網絡。
  5. 讓有世界存檔/配置/網絡更好的人開啓一個單人遊戲世界。
  6. 按下 Esc 並按下選項。
  7. 點擊向局域網開放按鈕。
  8. 設置選項並開啓局域網連接。
  9. 在聊天窗口會顯示5位數的端口號，把這個號碼發送到 Hamachi 的聊天窗口或者在其他渠道發送這個號碼。
  10. 讓非主機玩家轉到多人遊戲，並添加一個服務器或直連。
  11. 讓非主機玩家複製主機的 IPv4 地址(不是 IPv6)，它的樣子應該是這樣 ##.##.##.## 或 ###.###.#.#(不是 ####:#Δ::####:# Δ 代表一個字母)
  12. 讓非主機玩家把IP粘貼到服務器地址欄並添加 : 後面的5位數(看起來應該是 ###.###.#.#:##### 或 IPv4:端口號)
  13. 讓非主機玩家加入遊戲，你就可以和你的朋友一起玩耍了。

局域網的漏洞


Minecraft 1.5.2 的屏幕截圖 - 聊天文字為 「Local game hosted on 0.0.0.0:56646」

通過下列操作，你可以跳過該漏洞。把你「向局域網開放」獲得的端口號碼與 Hamachi IP合併。如 5.5.31.32 + 0.0.0.0:76152 = 5.5.31.32:76152。合併後的 IP 是正確的，你可以把這個 IP 發送給別人了。

要尋找你的 Hamachi IP，打開 Hamachi 並複製框框 / 前面的 IP。

要獲得你的端口號，在 Minecraft 按下 ESC 然後點擊「向局域網開放」，然後點擊「開始局域網世界」，會在聊天窗口顯示像上圖的消息。你也可以按下/，然後輸入「publish」，按下↵ Enter來開放局域網世界。
