音效索引

位置

Minecraft 的 Sound 文件夾在這些位置里：

  • Windows XP/Vista/7/8：%AppData%\.minecraft\assets 或 %AppData%\.minecraft\assets\virtual\legacy
  • Linux：~/.minecraft/assets
  • OS X：~/Library/Application Support/minecraft/assets

1.7 及以上的音效文件結構

   "sounds/music/menu/menu1.ogg": {
     "hash": "c157c56846f0e50620f808fecd9d069423dd6c41",
     "size": 1744657
   },

從這段文字我們可以看出 menu1.ogg 文件被標記為 c157c56846f0e50620f808fecd9d069423dd6c41 ，在 assets 下面的 objects 文件夾搜索那個標記的文字，就可以找到對應的文件。文件名的頭2格字符對應文件所在的文件夾，這可能會使搜索變得更快。定位後，你會發現這個文件沒有擴展名，但是仍然可以使用媒體播放器進行播放。如果你在使用其它系統，那麼你可以使用對應系統上能播放 .ogg 文件的媒體播放器播放這個文件。如果媒體播放器不能播放沒有擴展名的文件，那麼就在文件名後面加上 .ogg。

警告

如果你意外編輯或移除原始文件夾裡面的文件，你將不得不要刪除 Minecraft 根目錄裡面的 assets 文件夾，啓動器會在你下次啓動遊戲時自動下載它。如果你更改了音樂之後並連接到互聯網，啓動器就會自動下載原始文件，並用這些文件覆蓋被更改的文件。因此如果你想要保存你所作的更改，在遊玩前必須斷網。當然，你也可以創建一個包含你的音樂的自定義資源包。

Legacy 的子文件夾

在 .minecraft\assets\virtual\legacy\sounds 裡面有13個子文件夾。

  • Ambient：下雨/雷電天氣的洞穴音效。
  • Damage：玩家受傷音效。
  • Dig：破壞方塊。
  • Fire：火。
  • Firework：煙火音效。
  • Liquids：水/熔岩製造的聲音。
  • Minecart：由移動的礦車發出的聲音。
  • Mob：Minecraft 里生物的聲音。
  • Music：C418 製作的背景音樂。
  • Random：從吃食物到爆炸的幾種音效。
  • Records：Minecraft 里音樂唱片對應的音樂。
  • Step：腳步聲。
  • Tile：活塞。
