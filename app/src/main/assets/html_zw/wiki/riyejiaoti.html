日夜交替


一個完整的日夜交替過程。

日夜交替(Day-Night Cycle) 是主世界中2個主要亮度設置的交替過程，需時20分鐘。

時間換算


Minecraft時間與現實時間的大約換算圖

一天

白天


一天的開始。

白天是一天週期中最長的一節，歷時10分鐘。

單人遊戲中玩家在一個新世界首次出生時，時間總會處於白天的開始(0:00)。大多數多人遊戲服務器的時間也是從這個時間開始，但日夜交替會持續進行，不會受玩家加入影響。

白天時，太陽會逐漸升到藍天的中央。天空的具體顏色取決於當前所在的生物群系。例如，沙漠的天空比草原藍色更淡。

在將近正午的時候，所有直接暴露於天空下的方塊均被亮度15的陽光照射，此為亮度的最大值。陽光提供了樹苗、草和農作物生長所需的充足陽光。直接暴露在陽光下的大部分不死生物(包括僵屍和骷髏，不處於陰涼處、水中，且不戴頭盔)會被點燃。另外，陽光會使尚未追逐玩家的蜘蛛進入中立狀態，使末影人靠近玩家時隨機傳送並保持對玩家的中立。

在白天持續期間，陽光傳感器的信號輸出強度保持最大。

日落


在高處看到的日落。

日落是介於白天和夜晚之間的時間段，持續1 1⁄2分鐘。

日落時，太陽從西邊降落，月亮從東邊升起。下落的太陽附近的天空會有紅暈。日落時的太陽看上去還會漸漸變大。

所有直接暴露在天空之下的、未被其它光源照亮的方塊，每10秒鐘亮度降低1級。

在日落和日出之間的時間段，玩家可以睡在床上。在單人遊戲中，睡覺會直接將日夜交替快進到日出，且將天氣轉為晴朗。多人聯機中也可以實現，只需要所有的在線玩家都睡在床上即可。然而，遊戲中許多機制中時間並不會隨之流逝：比如，農作物不會生長，熔爐中的物品燒煉進度也不會改變。

夜晚


凍原的夜晚；可以看到多種生物。

夜晚比白天略短，持續7分鐘。

夜晚時，月亮會逐漸升到天空的中央，黑藍色的天空布滿了微小而白亮的星星。星星會隨著月亮移動，日落即將結束的時候會先於月亮出現。

夜晚期間，整個世界被黑暗掩蓋，亮度最低降至4，從而攻擊型生物會在地表生成。允許攻擊型生物生成的精確時間為9625刻。雖然夜晚的亮度低，但農作物仍會生長，除非玩家睡在床上度過夜晚。

在夜晚持續期間，反向陽光傳感器的信號輸出強度保持最大。

日出


太陽從地平線上升起。

日出是介於夜晚和白天之間的時間段，持續1 1⁄2分鐘。

日出時，月亮往西方地平線方向降落，太陽從東方地平線上升起。升起的太陽附近的天空會有紅暈。升起的太陽看上去會漸漸變大，但最後會隨著高度增加而慢慢縮小至8×8像素。

所有直接暴露在天空之下的方塊每隔10秒亮度增加1。最終，陽光的威力再一次展現，大部分怪物的威脅被驅除。

日出的曙光開始於22009刻，或現實時間的4:00:32。精確的停止刷怪時間為22812(04:48:43)，雨天則推遲至23031(05:01:08)。

鐘


鐘可使玩家獲知Minecraft時間。玩家在地下作業、看不到日夜交替現象時，鐘會發揮很大作用。它能避免玩家不小心在夜晚返回地面而成為敵對生物的獵物。

下界和末路之地沒有日夜交替，鐘也只能隨機地旋轉，不能發揮作用。

命令

如果開啓 命令功能，則可以使用/time命令修改時間，如下所示。要禁用或啓用日夜交替機制，可使用命令/gamerule doDaylightCycle [true/false]。

設置時間

可用/time set命令將時間設置為特定值。

語法：/time set <時間值 | day | night>

示例：

/time set 0 - 將時間設為日出。
/time set day - 將時間設為1000刻。
/time set 6000 - 將時間設為正午。
/time set 12000 - 將時間設為日落。
/time set night - 將時間設為13000刻。
/time set 18000 - 將時間設為午夜。

月相

遊戲中每過一天，時間計數便會增加24000刻。雖然每天的交替是一樣的，但月亮會經歷8種月相。雖然沒有命令直接更改月相，但/time add 24000命令可以快進至下一個月相。進一步而言，使用以下命令可以直接指定不同的月相：

/time set night - 滿月
/time set 38000 - 虧凸月
/time set 62000 - 下弦月
/time set 86000 - 殘月
/time set 110000 - 新月
/time set 134000 - 娥眉月
/time set 158000 - 上弦月
/time set 182000 - 盈凸月
