將遊戲數據保存到 Dropbox 雲端(僅限地圖數據)

該頁面的內容不受Mojang AB、Minecraft Wiki、Minecraft IRC頻道與Minecraft論壇的官方支持。

警告
因為這是 Wiki 的一個條目，可以在任何時間內被任何人編輯，因此建議你不要完全使用這個腳本，而是將其當作編寫腳本的指導手冊看待。

這裡是一個如何把遊戲數據上傳到雲端的教程，以便你可以在任何地方遊玩該存檔。

注意

該教程假設你只想上傳 Minecraft 存檔文件夾。如果你希望分離不同系統的文件夾(如 Minecraft/Linux、Minecraft/Windows，Minecraft/Toaster 等等)，那麼你就要在 Dropbox 里創建它們。本教程只會同步遊戲存檔，其它的 .jar 自定義文件(例如 Mod)或材質包則不包括。如果你不知道它們是什麼，那麼你的文件可能是安全的。

下載和安裝 Dropbox

第一步就是下載 Dropbox。Dropbox 會自動同步服務器的內容，允許你在線或通過PC客戶端訪問這些內容。Dropbox 是跨平台的，再加上可以通過網頁或手機網頁進行訪問。

前往 http://www.dropbox.com，註冊一個免費賬戶，下載應用程序並安裝它。你會自動獲得2GB的免費雲存儲空間。

安裝了 Dropbox 後，你需要知道 Dropbox 的文件夾位置：

  • 在 Windows XP 里，Dropbox 文件夾在用戶的「My Documents」文件夾里。引用該文件夾最簡單的方法就是使用環境變量，例如： "%HomeDrive%%HomePath%\My Documents\Dropbox"。因為「My Documents」含有空格，因此變量最前面和最後面的雙引號是必需的。
  • 在 Windows 7 和 8 里，默認 Dropbox 文件夾在 C:\Users\用戶名\Dropbox。使用環境變量 %HomeDrive%%HomePath%\Dropbox。
  • 在 Linux 里，默認位置在用戶的 home 文件夾。例如在 Ubuntu，其文件夾就在 ~/Dropbox
  • 在 Mac OS 里，其位置也應該在用戶的 home 文件夾。

無論你使用哪個系統，Dropbox 都有對應的非常好的文檔，可以協助你查找幫助：http://www.dropbox.com/help

複製你的存檔數據到 Dropbox

下一步，我們將會需要備份一份 Minecraft 的存檔數據，並且把這個備份放到 Dropbox 雲端。要完成本教程，我們將會複製 .minecraft 裡面的 saves文件夾，並把這個複製的文件夾放進 Dropbox 文件夾裡面的 Minecraft。你也可以選擇把它存在你選擇的 Dropbox 位置里。

Windows

在 Windows 里，你的 Minecraft 數據文件夾位於你的 Application Data 文件夾。你可以通過按下 Windows+R 或點擊 開始菜單 > 運行並在框里輸入 %AppData%\.minecraft。這將會為你打開 minecraft 文件夾。複製名為 saves 的文件夾，我們將會在 Dropbox 文件夾中創建 Minecraft 文件夾，然後把數據儲存在這裡面。

一旦這些文件複製完成後，回到你的 .minecraft 文件夾，然後重命名文件夾為 saves-backup。該步驟可確保你在出現不可恢復的錯誤時擁有一個備份，以及允許在最後的一個步驟中創建正確的符號鏈接。

Mac OS X

在 Mac OS X，Minecraft 數據文件夾位於您的用戶名 > Library > Application Support 文件夾。你可以通過打開 Finder → Go → Go To Folder → 輸入 ~/Library/Application Support/minecraft 來快速訪問該文件夾。要為接下來的引用作準備，最好的方法是為該文件夾製造一個別名，並將它複製到您的桌面。

現在，複製文件夾到 Dropbox 其中一個位置。最好重命名原始存檔文件夾(在您的 minecraft Application Support 文件夾里)為 saves-backup 作為備份，以防止發生意外情況時存檔文件夾出現問題。它也允許我們在接下來的一個步驟里創建鏈接到正確位置的符號鏈接。

Linux

在 Linux 里，您的 Minecraft 數據文件夾位於您的 user home 文件夾里。然而，這個文件夾可能在視圖里被隱藏了。如果您正在使用 GUI，確保已選擇 View > Show Hidden Files。

把存檔文件夾複製到您的 Dropbox。與之前提到的一樣，本教程我們在 Dropbox 創建了一個叫作 Minecraft 的文件夾來保存這個存檔文件夾。

一旦保存完成後，重命名存檔文件夾為 save-backup。

如果您擁有強大的命令行解析能力，這裡是上述步驟所使用的命令行。

if [ ! -e ~/Dropbox/Minecraft]
then
mkdir ~/Dropbox/Minecraft
fi
cp -r ~/.minecraft/saves ~/Dropbox/Minecraft/saves
mv ~/.minecraft/saves ~/Dropbox/saves-backup

創建符號鏈接

這是本教程的最後一個步驟，它將會允許用戶在使用他們文件的雲端版本。

Windows

GUI

一個免費的應用程序 Link Shell Extension
http://schinagl.priv.at/nt/hardlinkshellext/hardlinkshellext.html#download
讓在 Windows 上創建符號鏈接變得更容易，它允許你在資源瀏覽器里通過右鍵創建符號鏈接。

注意：如果您正在運行 Windows XP，您需要安裝 Server 2003 resource pack 以開啓符號鏈接，這個 Resource pack 可以在 這裡下載。

  1. 移動你的 Minecraft 存檔文件夾(位於 %appdata%\.minecraft)到您的 Dropbox。
  2. 右擊該新文件夾，並選擇 Pick Link Source。
  3. 回到您的 Minecraft 文件夾原始位置，然後在空白位置右擊，並點擊 Drop As… → Junction。
  4. 至此，您已經完成了把遊戲數據推送到雲端的全部設置。

命令提示符

  1. 右擊 cmd.exe 然後點擊以管理員身份運行。
  2. 在 cmd.exe 鍵入： cd %appdata%\.minecraft\
  3. 然後鍵入：mklink /d "saves" "<dropbox_path>\Minecraft\saves"
  4. 如果一切成功，那麼控制台就會輸出「Symbolic link created」。

Mac OS X

GUI

一個免費的應用程序 SymbolicLinker
http://seiryu.home.comcast.net/~seiryu/symboliclinker.html
允許你在 Finder 里通過右鍵創建符號鏈接。

  1. 下載並安裝 SymbolicLinker。
  2. 安裝後，移動您的 Minecraft 存檔文件夾(位於 '~/Library/Application Support/minecraft)到您的 Dropbox。
  3. 右擊該新文件夾並選擇 Make symbolic link。這將會在同一個文件夾創建符號鏈接。
  4. 把這個文件夾移到原來的位置(確保與原始文件夾同名)。
  5. 至此，您就完成了把遊戲數據導入雲端的全部設置。

Terminal

  1. 打開 Terminal
  2. 輸入：ln -s ~/Dropbox/Minecraft/saves ~/Library/Application\ Support/minecraft/saves

如果您移動了您的 Dropbox 到某個位置而不是默認位置，那麼您就需要修改第一個路徑。

Linux

GUI

打開2個文件瀏覽器窗口，一個窗口展示的是 Dropbox 文件夾，另外一個展示的是 minecraft 文件夾。
之後把 saves 拖到 Dropbox 文件夾那裡，然後按下 Alt 鍵拖回去到原來的位置。這應該會創建一個符號鏈接。

Terminal

要在 Terminal 裡面創建符號鏈接，在 Terminal 里輸入：
ln -s ~/.minecraft/saves ~/Dropbox/Minecraft
現在，當您啓動 Minecraft 時，您所有保存的世界應該都會在那裡了。
