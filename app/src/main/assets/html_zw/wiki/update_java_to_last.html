更新Java到最新版本

更新是遊玩Minecraft時最重要的事情之一, 因為有些技術性漏洞只能通過更新Java來解決。

從啓動器版本1.6.11開始，Minecraft啓動器和Java版本1.8.0_25綁定，其被默認使用。可以從個人資料的選項菜單來更改Java版本。

什麼是 Java？

Java是一門編程語言和一個運行平台。不像其它的如C++的語言，Java並不是直接運行在硬件上，而是運行在一個叫作JVM的虛擬機里。 Minecraft就是用Java編寫的，遊戲的所有邏輯都是使用它編寫的。

截止至2017年6月6日，最新的Java穩定版為 1.8.0_131，最新的開發版為 1.8.0_152-b04 和 1.9.0_b172。

注：Java 9並不兼 Minecraft，除非你在JVM啓動條件中移除-XX:+CMSIncrementalMode。

為什麼更新？

最新版本的Java包含了重要的增強功能以增強性能、穩定性以及運行在你的機器上的Java程序的安全性。安裝最新的Java更新將會確保Minecraft繼續安全高效地運行。

  • Minecraft1.12需要安裝Java 8來啓動。
  • Minecraft在非最新版本的 Java 上運行有時候會崩潰。
  • 更新會修復許多的問題和漏洞，通常情況下會帶來性能的提升。

下載地址

可以在 http://java.com/en/download/manual.jsp 獲取Java最新穩定版。

你也可以在 https://jdk8.java.net/8/ 獲取Java 8最新開發版。

你也可以在 https://jdk9.java.net/9/ 獲取Java 9最新開發版。

注意：開發版的Java是不穩定的，可能會導致意料之外的漏洞和崩潰。

備注

  • 由於Minecraft啓動器已和Java綁定，除非你使用的是第三方啓動器，否則你不需要單獨下載Java。
  • 如果你的機器是64位的，建議安裝64位的Java以獲得最佳性能。
  • 在更新之前卸載機器上所有版本的Java是不必要的。
  • 當你安裝最新版本的Java時，它會詢問是否需要卸載舊版本的Java，或自動卸載。
  • 有些系統需要管理員權限來安裝Java。
