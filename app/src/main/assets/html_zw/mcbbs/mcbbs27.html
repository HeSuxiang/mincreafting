常用指令大全

不少新人經常問我某某指令怎麼用，某某指令是什麼

那今天就教大家各種指令

死亡不掉落

/gamerule keepinventory true

防生物破壞地形

/gamerule mobGriefing false

修改難度

/difficulty 數字（1、簡單，2、普通，3、困難）

給予玩家某個物品

/give 玩家名 物品名

殺死某個玩家

/kill 玩家名

修改模式

/gamemode 數字 （0、生存模式，1、創造模式，2、冒險模式）

給予某個玩家所有權限

/op 玩家名

取消某個玩家所有權限

/deop 玩家名

傳送到某個玩家身邊或者傳送到某個坐標

/tp 玩家名或者坐標

（如果玩家名是中文，並且該指令無效的話輸入/tp 「玩家名」即可）

調整當前的時間

/time set 時間數（0-18000）

關閉火焰蔓延

/gamerule doFireTick false

關閉生物掉落物

/gamerule doMobLoot false

關閉生物生成

/gamerule doMobSpawning false

關閉方塊被破壞掉落物品

/gamerule doTileDrops false
