地图下载

在玩家的游戏中生成的地图可以被其他玩家下载并进行游玩。

地图存档的存储位置取决于其所使用的操作系统。目前选择 .minecraft 数据文件夹是不可能的，因此世界存储在一个已建立的文件夹里。

在 Beta 1.3 之前有5个地图存档位置，并且不能自定义它们的名称。为了使用5个以上的世界，就要重命名或移动文件夹。这不再必要了，因为理论上可以拥有不同数量的世界，每个世界都可以拥有它们自己的名称。

Windows

先决条件

一个压缩软件，例如 WinRAR 或 7-Zip。一些文件如 .RAR 不能在没有压缩软件的情况下进行解压，而在没有压缩软件的情况下可以处理 .ZIP 文件。

获得世界存档

使用网络浏览器（如 IE浏览器、Chrome 浏览器或火狐浏览器）下载存档。

解压文件

如果你下载下来的文件是 .ZIP 或 .RAR 文件的话，那么就有必要进行这个步骤。

  • 解压文件。右键，然后选择解压。如果文件格式是 .RAR，那么 Windows 将不能够打开它，这是你就需要一个压缩软件如 WinRAR 或 7-Zip。
  • 如果你看见一个叫做data的文件夹和一个叫做session.lock的文件，你就可以继续进行下一步了！将文件夹解压到你的桌面。
  • 如果看不见，那么文件可能存在，不过你得挖得更深才能看见。一些存档可能拥有2个文件夹，叫做Mac OS X 和 your_save。检查每个文件夹，直到你看见data文件夹和session.lock文件。一旦找到了，就将文件夹解压到你的桌面。

将文件放入 Minecraft 存档存储位置中

现在我们拥有了文件，我们将要把它们放到 Minecraft 的存档文件夹里。

  • 复制在桌面上的文件夹（复制data的上一级目录和session.lock，例如，如果data文件夹和session.lock存储在 your_save_multi.RAR 的Mac OS X的your_save中，那么就复制your_save文件夹。）
  • 按下 ⊞ Windows + R 打开运行窗口，并输入 %appdata%\roaming\.minecraft\saves ，然后点击 ↵ Enter，你将会进入 Minecraft 的 saves 文件夹，然后你就会留意到你的世界也在这里。
  • 在saves文件夹中按下 CTRL + V 以粘贴文件夹。
  • 检查your_save文件夹，确保文件夹里存在下面的文件/文件夹。
    • data
    • DIM-1
    • DIM1
    • players
    • region
    • level.dat
    • levl.dat_mcr
    • level.dat_old
    • session.lock
  • 一旦确定它们都在这里了，关闭 .minecraft 文件夹，并启动游戏。

在 Minecraft 中寻找存档

这里是最难的部分：在你的储存的所有存档中寻找你的新存档。

  • 启动后，点击“单人游戏”。
  • 你复制的文件夹明将会显示在世界名称的下面。大多数存档会叫做“world”。
  • 新存档通常都在最下面，但是也有例外，你将会需要寻找存档。

找到存档后，加载它并开始游玩吧！

Mac

寻找世界存档

从浏览器（如 Safari、Chrome 浏览器或火狐浏览器）中下载存档。

解压文件

（有时候下载的文件是没有被压缩的。检查文件是否为 .zip 或 .rar。）

  • 双击（或右击并选择打开）以解压文件夹。
  • 你将会看到在解压的文件夹里有一个文件夹。

将文件放置到 Minecraft 存档位置中

将文件夹移动到 /Users/[USER]/Library/Application Support/minecraft/saves。
如果你的 minecraft 文件夹在不同位置，使用那个不同的位置代替。

解决

如果在 Minecraft 中找不到对应的存档，请：

  • 检查你是否将一个文件夹放到 saves 文件夹里。
  • 检查你是否忘记解压文件夹。
  • 检查 Minecraft 是否使用了不同的 minecraft 文件夹。

Linux

该教程基于 Ubuntu 14.04.1 LTS 编写。其他的发行版本可能使用不同的资源浏览器，但是 .minecraft 的位置都是相同的。

先决条件

一个压缩软件，如 Archive Manager （文档管理器）。一些文件如 .RAR ，将在不能够在没有解压的情况下打开。ZIP 文件可以被 Archive Manager 打开。
