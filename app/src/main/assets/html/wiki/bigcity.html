大都市

如果你拥有一堆钻石，多于15组的木板和木头，一个专门存储圆石的巨大储存室，甚至打败了末影龙。已经拥有了信标？已经做了一个怪物磨床？你不知道你现在要做什么？那就尝试建造一座大都市吧！大都市拥有人和建筑物。要建造大都市，你需要一块足够大的平原。在一般的世界类型中，通常会生成一些平原（如草原和热带草原），但如果你要建造大都市，你就需要用到超平坦世界。但是，你也可以使用 Mod ，插件，或其他软件如 WorldEdit, MCEdit, VoxelSniper, BuildCraft 和 WorldPainter 清理陆地，创建一个大面积的平原，或者你可以用超平坦的预设去建造你的大都市。推荐在创造模式下建造大都市。

建造大都市总得要有个开始吧，从哪儿开始呢？嗯，由你自己决定！

前期准备

开始建造城市之前，你需要找到一块地方去建造它。一片森林或带有一些树的平原可能是一个好地方。在这个列表中，你将会需要一个司机，甚至要建造一片大海，因此将城市建在海边是一个不错的主意。实际上如果你想让这个都市在生存模式中发挥作用，你的第一栋建筑就是信标金字塔，因为它可以提供速度和速掘等加成。 注：如果你的服务器很少人，尝试邀请你的朋友，让他们作为这座城市的市民，或使用村民刷怪蛋。此后，这座城市看起来就不像是一座“鬼城”了。 你可以为“市民”选择游戏模式。实际上，玩家通常使用生存模式和冒险模式。如果是这样，玩家们应获得初始货币——32个绿宝石（城市货币），皮革护具（可染色），一组食物和一组铁（石）工具（1.8后冒险模式可以设置破坏哪些方块（甚至是正常情况下无法破坏的），用命令获取）。 要开始建造你的城市，你需要为第一个建筑物选定一个位置。任何类型的建筑物都将会被建造，但是你可能想让这个建筑物建在城市中心，因此当你建造这个建筑时，请将这个建筑物在城市里的布局记在脑海里。（如果你的城市很大，可以记在纸上） 注：如果你在生存模式中建造大都市，很有可能你需要一个 Inventory Editor。城市的规模决定于这个城市要容纳的人口。 如果你在建设一个指定风格的小镇，你可能会使用 MCEdit 去生成一些自定义交易的村民。

推荐建筑材料

推荐使用下列材料作为建筑材料：
  • 所有石质材料（圆石，石砖等）
  • 羊毛（地毯，人造草坪等）
  • 木头（建筑内部结构）
  • 木头（建筑外墙，楼层地板和支承结构）
  • 草方块，玻璃板，染色玻璃或染色玻璃板
  • 任何台阶
  • 六面相同的一种“双石台阶”（因为它和双石台阶方块名称相同）
  • 注:这种方块不使用第三方程序或命令是无法获得的。
  • 下界石英（可用大理石代替）
  • 陶瓦
  • 砖块或染色粘土
  • 基岩或黑曜石（放在墙的前后两侧，加强防护）
  • 注:同样需要第三方程序或命令。
  • 花岗岩，安山岩，闪长岩（无论是光滑的还是磨制的，用这些方块作为或小路（未很好的视觉效果，闪长岩可当作大理石使用）
  • 各种海晶石
  • 旗帜（告示牌或旗子）
  • 一些罕见又很难获得的方块如绿宝石块，金块和钻石块会使建筑物看起来很显眼，所以应该在需要引人注目的地方使用它们，比如说房屋的屋顶、奢侈的建筑和政府大楼。或者，如果你在使用 Mod，你可以使用红宝石，蓝宝石，紫水晶等等。注：在生存模式中保护这些罕见资源是重要的，因为它们很容易发现并通过铁镐挖到，这样会破坏建筑物甚至是城市的外观。

开始建设

你现在要开始建设你的城市了，下面的这些建议将会对你如何建设你的城市很有帮助，如果你对这些建议不满意，你可以想出另外一些主意。

是时候将你的建筑变成一个城市了，这个过程的快慢将取决于中央建筑的大小。如果你的中央建筑很大，那么在它周围的建筑要和它拥有相同的尺寸（但不要太大）。如果你的中央建筑很小，考虑为这个建筑添加更多的楼层。如果你计划建造一个非常巨大的城市，你就需要建造许多房子。这可能是最耗时的一个部分。

注：如果你在多人游戏中建设城市，尝试为这个城市建造一个合适的、像样的或一定规模的玩家基地。如果你要建设很大的城市，这个城市就要有300个以上的玩家和多种职业、房子。

住宅区

为了让住宅区看起来更好、更大和更有条理性，请这样做：

1. 从各个方向建造一条长长的道路（约30-200方块）到你的中央建筑。道路可以使用任何材料建造。推荐使用圆石，煤炭块，黑曜石，沙砾，石头和砂岩（在海边建造的城市适用），地狱砖或染色粘土（青色陶瓦是游戏中唯一一个带有平滑色彩的灰色方块）。你也可以使用黑色和黄色方块，使道路看起来像城市道路。

2. 为这些主干道添加一些支路，这些支路约8-20方块宽。

3. 在剩余的空间里，在道路的两侧建造房子。这些房子应该有一些家具，灯，床，或一个放着一些东西的箱子。如果你的游戏版本是或者高于红石更新的版本（1.5），你可以使用阳光传感器获得更好的效果。或者你可以“残忍”一些，不要给他们上述东西（他们将不得不去购买它！）。你可以建造一间复杂的房地产交易市场，只要你认为这可以改变你的城市，但这样意味着要进行大量的基础设施建设和需要一个强大的经济支撑。

4. 你可以建设不同密度的住宅区。低密度住宅区只能拥有1间屋子，中密度住宅区可以拥有1-10间房子，高密度住宅区可以拥有11间以上的房子。注：不要让你的低密度住宅区和你的秘密公寓靠的太近。你可以在距离公寓1条街道的地方开始建设。

一般建议：庞大的公寓建筑应该与商店“混”在一起，这样可以减少公寓到商店之间的行走距离，或者直接传送到商店。

注：让玩家在他们自己的土地上进行建设比你在一部分的土地上进行建设要好。

商业区

现在你已经有了一块供人居住的地方,检查一下是不是一切正常,然后开始建造商业区.

1. 在住宅区中选择一条道路，然后将这条道路延伸至少30个方块。

2. 在路旁边放置一个标志，然后继续将道路延伸50个方块。

3. 完成后，开始从标志那里建设支路，这些支路要远离住宅区，然后开始建造一些商店、饭店和其它服务业的建筑。

4. 商业区可以像住宅区一样拥有不同的密度。低密度商业区有1间商店，该商店的商品种类不多；中密度商业区有2-10间商店；高密度商业区销售的是高级商品，甚至是一个行业的产品销售区。注：建议将低密度和高密度商业区混合在一起建设。（一种这样做的途径是划分一片街道，该区域建设低密度商业区，然后在它们的后面建设高密度商业区。

一般建议：许多商业往来都和贸易有关。如果城市风格设定为中世纪，这种特点尤其突出。

工业区

1. 像商业区那样延伸道路。

2. 在道路的分叉处（不要太多）建设工厂和其它建筑。

3. 你可以拥有不用密度的工业区。低密度工业区通过高科技农场或低科技怪物磨床生产掉落物；中密度工业区要比低密度工业区更先进；高密度工业区通过高科技怪物磨床和植物农场大量生产物品

一般城市中工业区的建议：没有任何人会喜欢像烟雾缭绕的工厂，核废料容器，树皮尘土制造商，火力发电厂，巨大的蒸汽动力织布机，不稳定的科学实验室等一些会对环境造成严重污染的建筑。确保你的工业区不要靠近任何住着人的房子，最好不要让任何人在工业区中工作。在工业区中划分100个方块的绿化带或一条河。

注：如果你的城市属于敌对/战争城市，将工业区中的兵工厂和弹药库的面积扩建到最大，这样在敌人来犯时会有意想不到的效果。

农村地区

1. 像商业区那样延伸道路。

2. 在一个适合的地方建立一个农场或牧场。

3. 按照你的风格来建造房子。 (只有一个足矣，为了拥有更好的效果，你需要几个这样的房子)

4. 为这些房屋添加一些额外的东西（例如一些家具，地毯等）

5. 在路的尽头建立农场。

城市区

1. 做一条很宽的路，在上面放许多猪和马(用来当做车）。

2. 建设许多高层建筑。如果你的城市不是现代的或是有一个不可以建造摩天大楼的规则, 那么就不要把它们造的太高。

3. 在摩天大楼里建造办公室或一些公司。 如果你的城市是中世纪的，请忽略这部分。

4. 添加细节和绿化，在建筑里添加红石灯。 同上, 如果你的城市是中世纪的，请忽略这部分。

5. 如果需要的话，在你的城市中心建造一座雕像或像素画。

首都

1. 仿照城市道路的样子做一条路。

2. 建造重要的行政建筑，比如大使馆、市政府、求职中心与房地产中心。

3. 建造一些大型、豪华的住宅来供优秀的公民居住。

4. 建造非常重要的建筑, 如国会大厦, 并在首都建造一些有高价商品的商店。

5. 把一尊领袖雕像建造在这里。如果你愿意, 你可以用黄金制作它。你也可以使用一些绿色的方块建造它，如陶瓦，使它看起来像一尊青铜雕像。

6. 一个可以让高楼看起来很现代化的建议：摩天大楼最好采用许多玻璃、石英、双层石台阶、圆石、地狱砖和粘土建造。

7. 如果你想让城市看起来更现代化, 建立立交桥、旅游景点和博物馆。

8. 建造一个超大的（或比较大的）豪宅，没有人可以住进去，除非他们是市长、总统或皇帝。

注意: 最后一步你可以不做，这只是建议而不是命令。

码头

1. 建设一条通往海洋的道路。（或一条连接海洋的河流。）

2. 建设各种不同大小的码头，这些码头有船只停泊。同时建设人们的上船的位置（是实体，不是建筑物）。开设一些海洋商店（如鱼类交易市场）。

3. 如果你想，你可以建造一些军用船只。这些船只可以用于你的军事/海军基地。你可以在这条船上填满TNT大炮，并将它们朝向海洋。

4. 如果需要，建设一个船坞吧！你可以让人们在这里建设船只，如果你有 WorldEdit，你可以很容易的将船推下海。（或者你只需要将完成的和未完成的船只放在那里以供观赏）或者你可以安装mod来制作真正可行驶的船。

军事区

1.建设一条较宽的路，延伸到郊外。

2.建设一堵很高的城墙。（最好用很难破坏的东西来建造它，如黑曜石。或者在城墙里灌上熔岩也是一个不错的主意。）

3.建设许多高塔，并在城墙后面建设军营和仓库。

4.在高塔和城墙上部署许多弓箭手。如果你有更多武器mod,可以改为火枪手。

5.制造一些陷阱，比如TNT地雷。如果你有一些Mod，就可以把陷阱造的更强大。比如用暮色森林Mod中的喷火方块。
