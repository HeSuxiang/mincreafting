使用 Hamachi 进行局域网连接

该页面的内容不受Mojang AB、Minecraft Wiki、Minecraft IRC频道与Minecraft论坛的官方支持。

Hamachi 局域网

你可以设置一个 Hamachi 局域网，让处于 Hamachi 网络里的人通过 Hamachi 连接到你的局域网。

  1. 安装Hamachi。你要安装的是非托管版。
    https://secure.logmein.com/products/hamachi/download.aspx
  2. 在 Hamachi 创造一个 VPN 隧道。
  3. 让你的朋友进入你的 Hamachi 网络。
  4. 让服务器里的每个人把 Hamachi 网络设为家庭网络，或者允许 Minecraft 使用公共网络。
  5. 让有世界存档/配置/网络更好的人开启一个单人游戏世界。
  6. 按下 Esc 并按下选项。
  7. 点击向局域网开放按钮。
  8. 设置选项并开启局域网连接。
  9. 在聊天窗口会显示5位数的端口号，把这个号码发送到 Hamachi 的聊天窗口或者在其他渠道发送这个号码。
  10. 让非主机玩家转到多人游戏，并添加一个服务器或直连。
  11. 让非主机玩家复制主机的 IPv4 地址(不是 IPv6)，它的样子应该是这样 ##.##.##.## 或 ###.###.#.#(不是 ####:#Δ::####:# Δ 代表一个字母)
  12. 让非主机玩家把IP粘贴到服务器地址栏并添加 : 后面的5位数(看起来应该是 ###.###.#.#:##### 或 IPv4:端口号)
  13. 让非主机玩家加入游戏，你就可以和你的朋友一起玩耍了。

局域网的漏洞


Minecraft 1.5.2 的屏幕截图 - 聊天文字为 “Local game hosted on 0.0.0.0:56646”

通过下列操作，你可以跳过该漏洞。把你“向局域网开放”获得的端口号码与 Hamachi IP合并。如﻿ 5.5.31.32 + 0.0.0.0:76152 = 5.5.31.32:76152。合并后的 IP 是正确的，你可以把这个 IP 发送给别人了。

要寻找你的 Hamachi IP，打开 Hamachi 并复制框框 / 前面的 IP。

要获得你的端口号，在 Minecraft 按下 ESC 然后点击“向局域网开放”，然后点击“开始局域网世界”，会在聊天窗口显示像上图的消息。你也可以按下/，然后输入“publish”，按下↵ Enter来开放局域网世界。
