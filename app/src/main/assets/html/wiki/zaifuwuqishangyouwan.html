在服务器上游玩

Minecraft 是一个单人游戏（除非你在多人游戏上游玩）。你独自生存，并且你的朋友只有动物，而且你可能会把它们杀了来获得食物！你可能想和朋友一起漫步世界。只需和其他玩家和可以分享爬行者的更多生物一起体验。因此服务器将要在 Minecraft 世界的方块里做标记。这里是进一步的说明。

加入服务器

  • 在 Minecraft 服务器列表网站上寻找一个服务器。
  • 阅读服务器介绍并选择一个你喜欢的服务器。找到它的IP地址并把它复制到剪贴板（Windows系统使用:Ctrl+C；Mac系统使用:Cmd+C)。IP地址通常是一串数字，通常写在服务器名称的后面，或者是一个域名的后面。
  • 启动Minecraft，点击“多人游戏”后再点击添加服务器。 这里应有两个文字框,分别是服务器名称和服务器IP地址。 服务器名称并不重要，你可以随意输入。现在按下Ctrl+V(在Windows操作系统中)将服务器IP地址复制到框中。然后点击完成来保存服务器。
  • 此时你应返回到游戏中的服务器列表。服务器应该会在上面出现，右边会有绿色的条。
  • 点击这个服务器,再点击加入。正常来说，你将会被带到这个服务器。如果没有，下面列出了一些常见的错误来帮助你解决这个问题。

可能出现的错误

有时候事情会出错，因此这里有一些错误信息和他们代表的含义。点击加入服务器时可能会出现如下信息：

被服务器中断连接 或 过期的服务器！

如果Minecraft最近更新了，你可能会接收到这些错误信息。它们的意思是你运行的客户端版本与服务端的版本不同。你可以在启动器创建新的与服务端同版本的profile，然后等待服务端更新，服务端更新的时间取决于服主的反应。

你不在这个服务器的白名单列表上!

一些服务器开启了白名单系统来防止其他人加入服务器——通常的，如果要被加入到白名单，你不得不访问一些外部网站，通常是服务器的网站。

您的账号已被此服务器封禁！

您的用户名或IP已经被服务器封禁。通常意味着你做了一些违反服务器规则的事情。要解除封禁，你需要联系服务器的管理员以请求解封。如果一个你从来没有听说过的服务器封禁了你的 IP，意味着你的IP地址被封锁了。

被服务器中断连接。无效的会话(请尝试重启你的游戏)

当你登录到 Minecraft 时，你会获得一个 Mojang 服务器追踪会话 ID。你的账号只有最新的会话 ID 才被允许加入服务器（即使离线模式关闭）。按照这个消息去做，重启你的游戏。这可能是其他人登录过你的账号，为了防止这样的事发生，更改你的密码。你不能使用假冒的或破解的 Minecraft 连接服务器。

无法连接到服务器

该消息出现在服务器列表的旁边。这意味着你无法连接到这个服务器。这可能是因为服务器已经关闭，你没有网络连接，或者服务器响应超时（因为延迟）。

Cannot resolve hostname

该消息出现在服务器列表的旁边。这意味着找不到指定名称的服务器，检查地址并重试。

数据流终止

如果你试图以过快的速度连接服务器，就可能会出现这条消息；断开连接，等一会儿并尝试重新连接，这会有效的。

身份验证目前正在停机维护。

Minecraft 服务器检查会话 ID 的功能当前不可用。你可以在 help.mojang.com 的 Service Statuses 处查看服务器的当前状态

Java 错误，例如 java.lang.aMethodWhichReturnedAStackTrace(): 一个错误消息

该错误的发生原因取决于该错误的消息。在网络上搜索这个问题，你可能会找到一个解决方案。

缓冲区溢出

查看维基百科上的缓冲区溢出词条以获取更多信息。

在服务器上发生的事情

在 Minecraft 服务器上会存在多个原型，可以通过独特的游戏特性、规则和引入的结构来区分。2个服务器不会是相同的，并且原型之间的界线经常是模糊的或无法区分的。许多特殊类型的服务器会依靠服务端软件进行修改，例如 Bukkit 。服务端软件并不受 Mojang 的支持。Bukkit（以及其他服务端软件）服务器可能会带有修改原版服务器体验的插件。如果是 Bukkit 服务器，可以输入 /plugins。然后你就能搜索插件的用途了。

要查看常见的服务器类型，详见服务器页面。

对于未更改的服务端软件来说，这些是真的：

  • 你将会在服务器的生成点处生成。生成点默认有 16 保护区，在这片区域里你不能破坏方块或与物品进行互动。
  • 在屏幕的左下侧是聊天区域，人们会在这里发送信息。默认聊天按键为 T 键。你也可以向聊天框里输入命令，这与聊天内容是完全不同的事情。要查看可用的命令列表，输入 /help
