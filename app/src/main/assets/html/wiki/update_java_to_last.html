更新Java到最新版本

更新是游玩Minecraft时最重要的事情之一, 因为有些技术性漏洞只能通过更新Java来解决。

从启动器版本1.6.11开始，Minecraft启动器和Java版本1.8.0_25绑定，其被默认使用。可以从个人资料的选项菜单来更改Java版本。

什么是 Java？

Java是一门编程语言和一个运行平台。不像其它的如C++的语言，Java并不是直接运行在硬件上，而是运行在一个叫作JVM的虚拟机里。 Minecraft就是用Java编写的，游戏的所有逻辑都是使用它编写的。

截止至2017年6月6日，最新的Java稳定版为 1.8.0_131，最新的开发版为 1.8.0_152-b04 和 1.9.0_b172。

注：Java 9并不兼 Minecraft，除非你在JVM启动条件中移除-XX:+CMSIncrementalMode。

为什么更新？

最新版本的Java包含了重要的增强功能以增强性能、稳定性以及运行在你的机器上的Java程序的安全性。安装最新的Java更新将会确保Minecraft继续安全高效地运行。

  • Minecraft1.12需要安装Java 8来启动。
  • Minecraft在非最新版本的 Java 上运行有时候会崩溃。
  • 更新会修复许多的问题和漏洞，通常情况下会带来性能的提升。

下载地址

可以在 http://java.com/en/download/manual.jsp 获取Java最新稳定版。

你也可以在 https://jdk8.java.net/8/ 获取Java 8最新开发版。

你也可以在 https://jdk9.java.net/9/ 获取Java 9最新开发版。

注意：开发版的Java是不稳定的，可能会导致意料之外的漏洞和崩溃。

备注

  • 由于Minecraft启动器已和Java绑定，除非你使用的是第三方启动器，否则你不需要单独下载Java。
  • 如果你的机器是64位的，建议安装64位的Java以获得最佳性能。
  • 在更新之前卸载机器上所有版本的Java是不必要的。
  • 当你安装最新版本的Java时，它会询问是否需要卸载旧版本的Java，或自动卸载。
  • 有些系统需要管理员权限来安装Java。
