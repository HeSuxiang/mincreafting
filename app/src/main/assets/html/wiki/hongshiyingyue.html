这是一个教你怎样用音符盒和红石做音乐的教程。

谨记，音符盒上方要有一空格。

教程

  • 有很多方法来创造音乐，有些方法是用外部程序来弄的。
  • 你可以把你想的任何一首歌转到音符盒中，这个你可以用一个Davve做的应用程式Note Block Studio 来弄。
  • 首先，你需要歌曲的midi版。要是没有也没事，网上有很多格式转换器。
  • 然后，你应该把歌曲打开。这首歌应该是被你按了“Load”(下载)键的，然后找到你的.midi文件，再输入进来的。
  • 之后来听它听起来起来什么样的。注意，如果在右上角有提示，“Not compatible”(不兼容)，那你就要按那个按钮，点Fix(修复)来修复一下了。
  • 现在去“File → Export”(文件→导出)导出，然后在那选你自己想要的吧。
  • 当它正在输出下载 MCEdit的时候，会有程序把原理图输进你的Minecraft世界中。
  • 在MCEdit中把你的东西打开，然后把你刚刚输出的原理图随便找一个地方放，等它下载。
  • 按CTRL S来保存，关掉MC Edit，打开Minecraft。
  • ( Note Block Studio音域与节奏十分受限，只能仅仅用于参考，不建议将导出的效果直接作为成品使用)

小建议

  • 记住附近千万不能有水 ，不然你所有的红石都会被破坏。
  • 如果有很多红石的话，使用颜色代码和标签来标明不同的部分做什么是很有帮助的，以防它被破坏。
  • 你可以用底部的钢琴写你自己的音乐，也可以按放在其他方块后面的音符盒上的按钮来改变乐器。

常识

  • 敲一个音符盒或者给一个红石信号来发音。
  • 用放置方块按钮来改变音符盒的音高。
  • 注意红石粉会把它周围所有的方块都激活，不只有一个。

红石

通过一行行地放置红石中继器和音符盒能播放几次。另一个方法是把音符盒放在激活铁轨的侧面，你就可以在行驶的时候依自己的选择来播放音乐。

引申

  • 你可以用活塞来开始/停止歌曲。
  • 你可以用按钮替代拉杆来开关。
