Minecraft 1.6 及之后的版本

跨平台安装

  1. 转到 files.minecraftforge.net，如果使用下面的链接，跳到第2步。
  2. 找到你的 Minecraft(必须是 1.6+)对应最新或推荐的版本。下载链接在 Minecraft 版本的旁边，链接名叫“Installer”。
  3. 下载后，运行它(Windows)或在列表或文件夹中双击它(OS X 和 Windows)。Linux 用户需要手动运行 JAR 文件，使用如 java -jar <installerName> 这样的命令。
  4. 这将会打开一个叫做“Mod System Installer”的窗口。点击“Install Client”复选框。Th
  5. (可选)如果你想改变安装目录，点击显示着默认安装目录路径旁边的“...”按钮(.minecraft 文件夹)，然后选择一个你想安装的文件夹。
  6. 点击 Install 并等待安装完成。
  7. 打开新的 MinecraftLauncher ，选择“Forge”Profile，或点击“Edit Profile”并改变版本为“release x.x.x-Forgex.xx.x.xxx”(“x”为构建编号，例如1.6.4-Forge9.11.0.884)。

安装 Minecraft Modinstaller

Minecraft Modinstaller 能快速、容易地安装 Mod。你可以在开发者网站http://www.minecraft-installer.com 下载它。工具推荐 Mod 会根据你选择的 Minecraft 版本安装的 Mod 进行推送。全高清图像和评论描述会帮助你进行选择。目前，已超过150个 Mod可供下载。

  1. 在列表里选择你的 Minecraft 版本。
  2. 在左侧的列表寻找你喜欢的 Mod。
  3. 点击“Add”箭头来在右边的列表进行选择要安装的项目。
  4. 点击“Install”(所有的 Mod 都会下载，包括 Minecraft Forge)，然后 Minecraft 将会启动。

Minecraft 1.5 及之前

这是旧版本 Minecraft 的 Forge API 安装教程。现在1.6之后允许使用“简单安装器”安装 Minecraft Forge。

Windows 安装

  1. 选择你想要的 Mod 并下载它们。通常，它们会说明 Mod 对应的版本——例如：mod - 1.7.2 jar。下载下来的文件将会是一个 Zip 或 Java 文件。
  2. 现在你需要下载 Minecraft Forge，链接在最下方。你需要下载 Mod 对应版本的 Forge。如果你下载了 1.7.2 Forge，你需要所有的 1.7.2 Mod。
  3. 如果你尚未启动过 Minecraft，运行它。
  4. 允许 Forge Client installer。
  5. 定位到你的 .minecraft folder。你可以点击开始菜单，然后在搜索框中输入%appdata%，点击 roaming，然后点击 .minecraft。XP 用户要使用运行来代替搜索。
  6. 定位到你最近创建的“mods”文件夹。
  7. 把你下载的 Mod 拖到文件夹里。
  8. 关闭文件夹，打开名叫 1.*.*-Forge**.**.*.**** 的新 Minecraft 版本(* 为版本号)。

Mac 安装

  1. 不要忘记 Force Update！随后关闭它。
  2. 通过 Finder 程序进入到你的 Library，并进入 Application Support 文件夹。
  3. 之后你会找到文件夹“minecraft”，转到 bin 文件夹，并用 The Unarchiver 程序解压 minecraft.jar 的文件。
  4. 安装 Forge 版，它应该会自解压。它包含几个 classes。
  5. 把新 classes 逐个 放置到解压文件夹中。你也可以删除 META-INF，如果你希望(推荐这么做)。不要忘记移除 ibxm 文件夹里的材质！
  6. 测试 Minecraft，如果有菜单弹出，你就成功安装 Forge 了！
  7. 在 finder 里添加你想添加的 Mod 到 mods 文件夹中。(或者 coremods 文件夹，如果它是一个 coremod)，然后尽情享受吧！
