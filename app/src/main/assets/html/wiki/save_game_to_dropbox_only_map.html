将游戏数据保存到 Dropbox 云端（仅限地图数据）

该页面的内容不受Mojang AB、Minecraft Wiki、Minecraft IRC频道与Minecraft论坛的官方支持。

警告
因为这是 Wiki 的一个条目，可以在任何时间内被任何人编辑，因此建议你不要完全使用这个脚本，而是将其当作编写脚本的指导手册看待。

这里是一个如何把游戏数据上传到云端的教程，以便你可以在任何地方游玩该存档。

注意

该教程假设你只想上传 Minecraft 存档文件夹。如果你希望分离不同系统的文件夹（如 Minecraft/Linux、Minecraft/Windows，Minecraft/Toaster 等等），那么你就要在 Dropbox 里创建它们。本教程只会同步游戏存档，其它的 .jar 自定义文件（例如 Mod）或材质包则不包括。如果你不知道它们是什么，那么你的文件可能是安全的。

下载和安装 Dropbox

第一步就是下载 Dropbox。Dropbox 会自动同步服务器的内容，允许你在线或通过PC客户端访问这些内容。Dropbox 是跨平台的，再加上可以通过网页或手机网页进行访问。

前往 http://www.dropbox.com，注册一个免费账户，下载应用程序并安装它。你会自动获得2GB的免费云存储空间。

安装了 Dropbox 后，你需要知道 Dropbox 的文件夹位置：

  • 在 Windows XP 里，Dropbox 文件夹在用户的“My Documents”文件夹里。引用该文件夹最简单的方法就是使用环境变量，例如： "%HomeDrive%%HomePath%\My Documents\Dropbox"。因为“My Documents”含有空格，因此变量最前面和最后面的双引号是必需的。
  • 在 Windows 7 和 8 里，默认 Dropbox 文件夹在 C:\Users\用户名\Dropbox。使用环境变量 %HomeDrive%%HomePath%\Dropbox。
  • 在 Linux 里，默认位置在用户的 home 文件夹。例如在 Ubuntu，其文件夹就在 ~/Dropbox
  • 在 Mac OS 里，其位置也应该在用户的 home 文件夹。

无论你使用哪个系统，Dropbox 都有对应的非常好的文档，可以协助你查找帮助：http://www.dropbox.com/help

复制你的存档数据到 Dropbox

下一步，我们将会需要备份一份 Minecraft 的存档数据，并且把这个备份放到 Dropbox 云端。要完成本教程，我们将会复制 .minecraft 里面的 saves文件夹，并把这个复制的文件夹放进 Dropbox 文件夹里面的 Minecraft。你也可以选择把它存在你选择的 Dropbox 位置里。

Windows

在 Windows 里，你的 Minecraft 数据文件夹位于你的 Application Data 文件夹。你可以通过按下 Windows+R 或点击 开始菜单 > 运行并在框里输入 %AppData%\.minecraft。这将会为你打开 minecraft 文件夹。复制名为 saves 的文件夹，我们将会在 Dropbox 文件夹中创建 Minecraft 文件夹，然后把数据储存在这里面。

一旦这些文件复制完成后，回到你的 .minecraft 文件夹，然后重命名文件夹为 saves-backup。该步骤可确保你在出现不可恢复的错误时拥有一个备份，以及允许在最后的一个步骤中创建正确的符号链接。

Mac OS X

在 Mac OS X，Minecraft 数据文件夹位于您的用户名 > Library > Application Support 文件夹。你可以通过打开 Finder → Go → Go To Folder → 输入 ~/Library/Application Support/minecraft 来快速访问该文件夹。要为接下来的引用作准备，最好的方法是为该文件夹制造一个别名，并将它复制到您的桌面。

现在，复制文件夹到 Dropbox 其中一个位置。最好重命名原始存档文件夹（在您的 minecraft Application Support 文件夹里）为 saves-backup 作为备份，以防止发生意外情况时存档文件夹出现问题。它也允许我们在接下来的一个步骤里创建链接到正确位置的符号链接。

Linux

在 Linux 里，您的 Minecraft 数据文件夹位于您的 user home 文件夹里。然而，这个文件夹可能在视图里被隐藏了。如果您正在使用 GUI，确保已选择 View > Show Hidden Files。

把存档文件夹复制到您的 Dropbox。与之前提到的一样，本教程我们在 Dropbox 创建了一个叫作 Minecraft 的文件夹来保存这个存档文件夹。

一旦保存完成后，重命名存档文件夹为 save-backup。

如果您拥有强大的命令行解析能力，这里是上述步骤所使用的命令行。

if [ ! -e ~/Dropbox/Minecraft]
then
mkdir ~/Dropbox/Minecraft
fi
cp -r ~/.minecraft/saves ~/Dropbox/Minecraft/saves
mv ~/.minecraft/saves ~/Dropbox/saves-backup

创建符号链接

这是本教程的最后一个步骤，它将会允许用户在使用他们文件的云端版本。

Windows

GUI

一个免费的应用程序 Link Shell Extension
http://schinagl.priv.at/nt/hardlinkshellext/hardlinkshellext.html#download
让在 Windows 上创建符号链接变得更容易，它允许你在资源浏览器里通过右键创建符号链接。

注意：如果您正在运行 Windows XP，您需要安装 Server 2003 resource pack 以开启符号链接，这个 Resource pack 可以在 这里下载。

  1. 移动你的 Minecraft 存档文件夹（位于 %appdata%\.minecraft）到您的 Dropbox。
  2. 右击该新文件夹，并选择 Pick Link Source。
  3. 回到您的 Minecraft 文件夹原始位置，然后在空白位置右击，并点击 Drop As… → Junction。
  4. 至此，您已经完成了把游戏数据推送到云端的全部设置。

命令提示符

  1. 右击 cmd.exe 然后点击以管理员身份运行。
  2. 在 cmd.exe 键入： cd %appdata%\.minecraft\
  3. 然后键入：mklink /d "saves" "<dropbox_path>\Minecraft\saves"
  4. 如果一切成功，那么控制台就会输出“Symbolic link created”。

Mac OS X

GUI

一个免费的应用程序 SymbolicLinker
http://seiryu.home.comcast.net/~seiryu/symboliclinker.html
允许你在 Finder 里通过右键创建符号链接。

  1. 下载并安装 SymbolicLinker。
  2. 安装后，移动您的 Minecraft 存档文件夹（位于 '~/Library/Application Support/minecraft）到您的 Dropbox。
  3. 右击该新文件夹并选择 Make symbolic link。这将会在同一个文件夹创建符号链接。
  4. 把这个文件夹移到原来的位置（确保与原始文件夹同名）。
  5. 至此，您就完成了把游戏数据导入云端的全部设置。

Terminal

  1. 打开 Terminal
  2. 输入：ln -s ~/Dropbox/Minecraft/saves ~/Library/Application\ Support/minecraft/saves

如果您移动了您的 Dropbox 到某个位置而不是默认位置，那么您就需要修改第一个路径。

Linux

GUI

打开2个文件浏览器窗口，一个窗口展示的是 Dropbox 文件夹，另外一个展示的是 minecraft 文件夹。
之后把 saves 拖到 Dropbox 文件夹那里，然后按下 Alt 键拖回去到原来的位置。这应该会创建一个符号链接。

Terminal

要在 Terminal 里面创建符号链接，在 Terminal 里输入：
ln -s ~/.minecraft/saves ~/Dropbox/Minecraft
现在，当您启动 Minecraft 时，您所有保存的世界应该都会在那里了。

为了链接任何其它您希望游玩的计算机，您必须遵循下列步骤 - 安装 Dropbox，然后创建符号链接。
