安装快照

安装快照可以了解到在接下来的 Minecraft 版本中的新内容，然而许多人不知道怎么安装快照，因此开设本教程。根据本教程，你将学会如何安装快照。

1.6 以后

为安装并开始一个1.6的快照版本，先下载启动器。这是可用的下载链接minecraft.net/download

默认的，你可以选择 Minecraft 的版本并使用该版本进行游戏（如1.5.2，1.5.4和1.7.2）。为了安装开发版本，请按照以下操作：

  • 为开发版本创建新的用户资料（Profile)。点击启动器上的 New Profile 按钮，将会弹出一个对话框。
  • 命名这个版本（如 'Snapshots'）。为防止不必要的麻烦，请尽量不要使用中文对该版本进行命名。
  • 点击 Game Directory 复选框，并在文本框后面添加一个文件夹路径（如 "C:\"）。
  • 点击 Enable experimental development versions(snapshots) 复选框。
  • 选择你喜欢的快照版本
  • 点击 Save Profile。

现在，你可以在启动器上使用最晚的版本和开发版本进行游戏。注意 Minecraft 不会检查数据的兼容性。当你进行开发时，你应该使用一个临时的世界进行开发。（请注意：有可能快照没有改变游戏目录，然而将地图从游戏目录中分离可以防止数据被破坏。）

如果你想比较旧的快照，点击启动器上的 Edit Profile 并在 Use versions 下拉选择框中选择一个版本。

1.6之前

为了安装 1.5 和较早的快照，请按照下列步骤：

第一步： 转到 快照历史 页面，选择快照版本并在右面的“下载链接”中下载快照。
第二步： 转到 .minecraft 文件夹。

  • Windows XP/Vista/7/8: %appdata%\.minecraft\（只需将这个地址复制到你的地址栏并点击回车）
  • Linux: ~/.minecraft/ (该文件夹可能在 “Home” 文件夹中隐藏）
  • Mac OS X: ~/Library/Application Support/minecraft/ （该文件夹可能被隐藏）

第三步： 使用你下载的快照替换 "minecraft.jar" 的 "bin" 文件夹，然后启动 Minecraft。

  • Linux 用户，如果想取消隐藏文件夹，请按 Ctrl+H 。

恭喜，你已经成功安装快照！

警告1: 打开已存在的单人世界存档并不是没有风险：该存档可能会被破坏或无法加载。如果你在新快照中加载地图并且这个新快照比当前版本要低，为了避免风险，请备份存档！（存档在 \.minecraft\saves\）

警告2: 如果你为你的 Mod 存档进行备份，那么你应该为 minecraft.jar 进行备份。只需将这个文件复制到一个安全的位置。（这个文件在 \.minecraft\bin）


如果想恢复到原来的版本，请在启动器的选项中点击 "Force Update"。
