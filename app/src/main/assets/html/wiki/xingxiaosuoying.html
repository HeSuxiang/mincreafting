音效索引

位置

Minecraft 的 Sound 文件夹在这些位置里：

  • Windows XP/Vista/7/8：%AppData%\.minecraft\assets 或 %AppData%\.minecraft\assets\virtual\legacy
  • Linux：~/.minecraft/assets
  • OS X：~/Library/Application Support/minecraft/assets

1.7 及以上的音效文件结构

   "sounds/music/menu/menu1.ogg": {
     "hash": "c157c56846f0e50620f808fecd9d069423dd6c41",
     "size": 1744657
   },

从这段文字我们可以看出 menu1.ogg 文件被标记为 c157c56846f0e50620f808fecd9d069423dd6c41 ，在 assets 下面的 objects 文件夹搜索那个标记的文字，就可以找到对应的文件。文件名的头2格字符对应文件所在的文件夹，这可能会使搜索变得更快。定位后，你会发现这个文件没有扩展名，但是仍然可以使用媒体播放器进行播放。如果你在使用其它系统，那么你可以使用对应系统上能播放 .ogg 文件的媒体播放器播放这个文件。如果媒体播放器不能播放没有扩展名的文件，那么就在文件名后面加上 .ogg。

警告

如果你意外编辑或移除原始文件夹里面的文件，你将不得不要删除 Minecraft 根目录里面的 assets 文件夹，启动器会在你下次启动游戏时自动下载它。如果你更改了音乐之后并连接到互联网，启动器就会自动下载原始文件，并用这些文件覆盖被更改的文件。因此如果你想要保存你所作的更改，在游玩前必须断网。当然，你也可以创建一个包含你的音乐的自定义资源包。

Legacy 的子文件夹

在 .minecraft\assets\virtual\legacy\sounds 里面有13个子文件夹。

  • Ambient：下雨/雷电天气的洞穴音效。
  • Damage：玩家受伤音效。
  • Dig：破坏方块。
  • Fire：火。
  • Firework：烟火音效。
  • Liquids：水/熔岩制造的声音。
  • Minecart：由移动的矿车发出的声音。
  • Mob：Minecraft 里生物的声音。
  • Music：C418 制作的背景音乐。
  • Random：从吃食物到爆炸的几种音效。
  • Records：Minecraft 里音乐唱片对应的音乐。
  • Step：脚步声。
  • Tile：活塞。
