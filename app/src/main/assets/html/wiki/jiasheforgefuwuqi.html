  • 启动服务器的命令: java -Xmx1024M -Xms1024M -jar minecraft_server.jar nogui

注意

如果启动服务器的命令不工作，请尝试把“minecraft_server ”重命名为“minecraft_server.jar”(不包括引号)。

或者你也可以使用这个命令: java -Xmx1024M -Xms1024M -jar minecraft_server.jar nogui


如果你收到“java无法启动”的消息， replace "java" with its file location. (例如: "C:\Program Files (x86)\Java\jre7\bin\javaw.exe")

完整的命令是:"C:\Program Files (x86)\Java\jre7\bin\javaw.exe" -Xmx1G -Xms1G -jar forge-1.7.2-10.12.2.1121-universal.jar nogui pause
