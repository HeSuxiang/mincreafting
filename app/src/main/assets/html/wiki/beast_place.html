最佳居住生物群系

对于庇护所来说一件最重要的事就是其位置。从本教程中你将会学习到在不同的生物群系建造住所的利弊。请注意本教程还未完善。

峭壁

优点

有时作为采矿的副产品你会采到绿宝石矿石。除了利于与村民交易，多山的环境将有利于某些计划的开展，例如开采大量石头和矿石；并且在这里会生成羊驼。

缺点

容易摔死，山底容易生成熔岩，难以建造平房，并有可能挖出怪物蛋并放出蠹虫。

实用资源

圆石，雪，冰，绿宝石，皮革。

这种生物群系适合在山体中居住，而不是在山体外建造房子。在这里可以很好地通过交易获取资源，因为如果它靠近一个村庄，你可以在开采得到绿宝石后与村民进行交易。在这里建房子很危险，因为容易摔死或是受伤。山与山之间的移动十分困难，但末影珍珠和桥可以起作用。在这里观赏自然景观很棒。大量山拱和悬崖在此生成。y坐标大于95的地方会开始下雪，可刷雪或刷冰。峭壁上会生成羊驼，这可以提供较多的皮革。

提示

用末影珍珠四处走动。

想避免蠹虫，开采石头时注意时间，如果挖掘时间很长，它就是怪物蛋。查看怪物蛋来获取更多信息。
如果你想在这里建造矿洞， 效率镐 以及2级速掘信标都有帮助。

沼泽

优点

史莱姆；沼泽小屋；有大量适合建房子的平地，大量食物，蘑菇煲。

缺点

如果不建桥或不使用船，穿越水域将非常慢；史莱姆和女巫都可以杀死你；怪物在这里容易生成；这里也没有更多的花做染料。

实用资源

粘土，木头，甘蔗，藤蔓，蘑菇，粘液球，兰花。

这种生物群系好在平坦，多水，多树，以及有大量不可再生的粘土。虽然存在女巫和史莱姆，但是你可以收集粘液球来制作粘性活塞 (在1.8版本中还可制作粘液块)。蘑菇通常在藤蔓下生成，可作为不错的初始食物资源：蘑菇煲。睡莲通常在湖中生成。 兰花只在这里有，也就是说你只能在这里采集到它们。有趣的是，火并不会在沼泽生物群系里扩散。此外，这里因多水所以居住起来十分安全(特别是防止恶意破坏)。

草原

优点

适于建房的平坦地形；大量的草丛可作为农场的初始种子来源。村庄、马、向日葵会在向日葵草原上生成；在刷怪的时候没有障碍物；有湖泊也有熔岩湖；也有高的台地能避开怪物们。村庄有几率生成铁匠铺，铁匠铺中会生成箱子。

缺点

也许会缺少树木；在夜晚的村庄中僵尸会袭击你。

实用资源

向日葵，种子，木头。

草原是玩家的理想生物群系。这里没有障碍物，所以走或跑起来都很快。在平坦地形上很容易建造大型住宅，而你可以与NPC村民可以来得到建造所需材料。马在这里生成，有着除热带草原外其他生物群系所没有的极快的移动速度。这里有大量草丛，可以得到最开始的一些种子来播种小麦(虽然NPC村民是更好的食物来源)。这里有很多沟谷、湖泊和矮丘。

冷针叶林

优点

雪；狼；冰；木头。

缺点

雪通常在你的庇护所上堆积；水会结冰；有大量的障碍物；层层的树木将会延缓怪物靠近你的速度，但也会减低你的速度；云杉树在地面上会因为太高而难以够到；而且狼会因为你的攻击而攻击你。

实用资源

雪，冰，木头，蕨类。

冷针叶林是一种寒冷多雪的生物群系。此处有许多雪、冰和树。狼在这里生成，而且如果你驯养他们，他们将会保护并爱上你。雪同样在这里生成，可以建造教程/刷冰。雪天代替了雨天，所以 教程/刷雪也行的。此处水的结冰影响了甘蔗的生成导致这里根本没有甘蔗。同理你的大量水资源也会结冰，有时很烦人。庇护所上方的落雪有时令人不悦，且高大的云杉并不利于采集(因为叶子比较少，掉落树苗的可能性更小)。此生物群系的变种，针叶林，虽然更不利于冰雪的采集(水不经常结冰且根本不下雪)，但是却没有冰和雪的困扰。

大型针叶林

优点

灰化土；苔石；狼。

缺点

树的树叶很少，难以采集树苗。

实用资源

灰化土，苔石，蕨类，木头。

大型针叶林是正常针叶林的一个变种，这里生成的巨大树木虽然树叶很少但却提供大量木头。没有草的泥土使这里看起来死气沉沉。但是，这里有很多在其他地方很稀有的蕨类以及灰化土，一种绝佳的菌丝替代物(因为蘑菇可以在白天放置)。苔石在此自然生成，所以你不需要剪刀来剪取藤蔓制作它。并且由于这是针叶林的变种，狼也在此生成！

沙漠

优点

沙漠神殿在此生成；地形平坦；沙漠村庄；有时会有暴露的地牢；仙人掌；大量的沙子(这意味着你能得到大量砂岩和玻璃及其制品)；末影人在这里比较常见(借以去往末地)；在沙漠神殿自然生成的TNT。

缺点

水源偏短缺(有时会有较小的水潭，还可能会有沙漠水井或村庄中的水井)；这里不会下雨，不利于钓鱼；除兔子外没有被动型生物(除非你把它们带进来或者它们自己移动进来)；沙子会下落，很危险；大量尸壳降低你的饥饿值。

实用资源

沙子；仙人掌；枯死的灌木；砂岩；TNT；玻璃和从沙漠神殿中获得的物质。

沙漠生物群系或许为大量玩家所厌恶。但是它确实不失为一个好地方。在这里可以获得大量的不可再生的沙子资源以及在其他地区罕见的仙人掌。水不在这里生成(若你在沙漠中央重生将会是一个大问题)。区块生成时不会有被动型生物生成，而且没有草也意味着没有动物生成。树也不在这里生成。但是许多高级玩家因为这里的沙漠景观以及他们所需要的砂岩而喜爱这种生物群系。虽然对新手来讲这不是最好的生物群系，但在别处建造完房子后这也是不错的下一个选择。

蘑菇岛

优点

菌丝；蘑菇；巨型蘑菇；在陆地上不会有敌对生物(除了僵尸围城)；另外你还可以建造蘑菇农场并通过碗来制作无限的蘑菇煲。

缺点

没有怪物意味着无法刷怪，也没有经验和掉落物；没有木头；很少与其他陆地相连。

实用资源

哞菇，菌丝，蘑菇。

这种生物群系十分稀有，且一般不与"大陆"相连。但其优点在于不论你在哪，怪物都不会在这里自然生成。地牢与废弃矿井依旧会有。这里对于新手玩家来说完全安全，对有经验的玩家来说也不错。这里唯一的生物，哞菇，用碗点击后可以获取蘑菇煲。所以这是个无限的食物来源！许多勇于挑战的玩家将找到蘑菇岛视为一项伟绩。这里的菌丝适合种植蘑菇，而这里也是两个自然生成巨型蘑菇的生物群系之一(另外一个是黑森林)。由于没有怪物，这里并不是最好的生物群系，也就是说无法建造刷怪塔。所以线，腐肉，火药，骨头以及所有怪物掉落物你都无法在此得到。树也不在此生成，所以没有木头来源。但是既然蘑菇岛如此稀有，找到后可以回到大陆，开采些木头与树苗后再赶紧回来。

森林

优点

大量木头；草色亮绿；小山高频率生成使得很容易欣赏到好风景。

缺点

房屋建造十分不便；怪物和地形会带来危险；因为有大量的树而难以跑动。

实用资源

木头，花。

这种生物群系对新手来说很不错，因为它提供大量木头，但树与树间没有太大的空隙而不能建造大型建筑，以至于需要砍伐一些树木。在得到很多木头后，一个高级玩家会搬出树林，移动到其他较平坦的生物群系来建造房子或堡垒。记住，对于新手玩家来说，这是一个危险的生物群系。树木会掩盖怪物，并使躲避爬行者更加困难。骷髅会朝你射箭，让你向它跑去，然后无情地杀死你。并且由于你大概不会有很多线，你很难用弓来还击。 不幸的是，森林及其变种生物群系中不会生成村庄之类自然建筑物，但会有地牢或要塞。

平顶山

优点

这种生物群系风景优美，并有大量的陶瓦以及稀有的红沙。而且特别方便你找到一个平顶山高原F变种，那里有森林，易于建造房屋。平顶山的废弃矿井有大量金矿，可以合成金苹果。在发生紧急状况时，总是能帮上忙。

缺点

这种生物群系中，树木十分稀少，并且由于地表大部分被染色粘土占据，岩层必须向下挖掘很久才能够被找到(除非是高原F变种)。

实用资源

大量彩色粘土。

冰刺平原

一个拥有随机冰刺的有趣生物群系。像平顶山一样美丽但也一样资源稀缺。但是如果它靠近一个资源丰富的生物群系，你倒是可以在此建造一个有趣的房子。在这建房更多是为了好看而不是实用。高高的冰刺稍加改造可以做成一个塔， 内部就形成了螺旋梯和顶部的一个房间。小一些的冰刺可以砍一些做成一个小屋。当然，这里其实不是个好地方。但会有流髑陪你，意味着你可以获得迟缓药箭。

深海

优点

大量墨囊，没有其他生物(除了鱿鱼)，海底遗迹在此生成，如果有睡莲将可以直接从水面开始建造。

缺点

缺少建造房子的地方，缺少食物(除非你有把握击败守卫者，这样可以获得生鱼)，并且如果你没有船或深海探索者附魔靴将很难移动，也有危险的守卫者。

实用资源

水、墨囊。

丛林

优点

大量树木，丛林神庙中有机会找到宝物(也可当作临时住宅)，还有豹猫，鹦鹉，可可豆，西瓜等特产。 很容易找到食物，由于树叶几乎铺满地表，怪物生成几率降低。丛林茂密的环境成为玩家房屋的天然掩护屏障，故此很少有玩家会深入丛林。

缺点

厚厚的树叶缩小了建造空间，从树上掉落会造成伤害，环境令导航十分困难，如果电脑配置不好，大量高大树木的加载会十分缓慢，且不容易行走。

实用资源

丛林木；藤蔓；从丛林神庙中所获得的物资；可可豆；树苗；西瓜。

虽然丛林并不适合新手开始生存，但是木头真的很多，因为丛林木是Minecraft中第二大的树木(第一是云杉木)。而且如果你能躲开其中的陷阱，丛林神庙中有大量物品可以拿走。 唯一的问题是树叶实在太多，所以极为不利于导航，有时会令人迷路。

热带草原

优点

树木丰富，会生成马，还会生成热带草原村庄；并且这里会生成羊驼。

缺点

水源的短缺使农场难以建设，会有高草丛挡住视野，所以请时刻留意周遭的怪物并定期除草。

实用资源

木头、马、村庄。
