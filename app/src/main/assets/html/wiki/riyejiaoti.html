日夜交替


一个完整的日夜交替过程。

日夜交替（Day-Night Cycle） 是主世界中2个主要亮度设置的交替过程，需时20分钟。

时间换算


Minecraft时间与现实时间的大约换算图

一天

白天


一天的开始。

白天是一天周期中最长的一节，历时10分钟。

单人游戏中玩家在一个新世界首次出生时，时间总会处于白天的开始(0:00)。大多数多人游戏服务器的时间也是从这个时间开始，但日夜交替会持续进行，不会受玩家加入影响。

白天时，太阳会逐渐升到蓝天的中央。天空的具体颜色取决于当前所在的生物群系。例如，沙漠的天空比草原蓝色更淡。

在将近正午的时候，所有直接暴露于天空下的方块均被亮度15的阳光照射，此为亮度的最大值。阳光提供了树苗、草和农作物生长所需的充足阳光。直接暴露在阳光下的大部分不死生物（包括僵尸和骷髅，不处于阴凉处、水中，且不戴头盔）会被点燃。另外，阳光会使尚未追逐玩家的蜘蛛进入中立状态，使末影人靠近玩家时随机传送并保持对玩家的中立。

在白天持续期间，阳光传感器的信号输出强度保持最大。

日落


在高处看到的日落。

日落是介于白天和夜晚之间的时间段，持续1 1⁄2分钟。

日落时，太阳从西边降落，月亮从东边升起。下落的太阳附近的天空会有红晕。日落时的太阳看上去还会渐渐变大。

所有直接暴露在天空之下的、未被其它光源照亮的方块，每10秒钟亮度降低1级。

在日落和日出之间的时间段，玩家可以睡在床上。在单人游戏中，睡觉会直接将日夜交替快进到日出，且将天气转为晴朗。多人联机中也可以实现，只需要所有的在线玩家都睡在床上即可。然而，游戏中许多机制中时间并不会随之流逝：比如，农作物不会生长，熔炉中的物品烧炼进度也不会改变。

夜晚


冻原的夜晚；可以看到多种生物。

夜晚比白天略短，持续7分钟。

夜晚时，月亮会逐渐升到天空的中央，黑蓝色的天空布满了微小而白亮的星星。星星会随着月亮移动，日落即将结束的时候会先于月亮出现。

夜晚期间，整个世界被黑暗掩盖，亮度最低降至4，从而攻击型生物会在地表生成。允许攻击型生物生成的精确时间为9625刻。虽然夜晚的亮度低，但农作物仍会生长，除非玩家睡在床上度过夜晚。

在夜晚持续期间，反向阳光传感器的信号输出强度保持最大。

日出


太阳从地平线上升起。

日出是介于夜晚和白天之间的时间段，持续1 1⁄2分钟。

日出时，月亮往西方地平线方向降落，太阳从东方地平线上升起。升起的太阳附近的天空会有红晕。升起的太阳看上去会渐渐变大，但最后会随着高度增加而慢慢缩小至8×8像素。

所有直接暴露在天空之下的方块每隔10秒亮度增加1。最终，阳光的威力再一次展现，大部分怪物的威胁被驱除。

日出的曙光开始于22009刻，或现实时间的4:00:32。精确的停止刷怪时间为22812（04:48:43），雨天则推迟至23031（05:01:08）。

钟


钟可使玩家获知Minecraft时间。玩家在地下作业、看不到日夜交替现象时，钟会发挥很大作用。它能避免玩家不小心在夜晚返回地面而成为敌对生物的猎物。

下界和末路之地没有日夜交替，钟也只能随机地旋转，不能发挥作用。

命令

如果开启 命令功能，则可以使用/time命令修改时间，如下所示。要禁用或启用日夜交替机制，可使用命令/gamerule doDaylightCycle [true/false]。

设置时间

可用/time set命令将时间设置为特定值。

语法：/time set <时间值 | day | night>

示例：

/time set 0 - 将时间设为日出。
/time set day - 将时间设为1000刻。
/time set 6000 - 将时间设为正午。
/time set 12000 - 将时间设为日落。
/time set night - 将时间设为13000刻。
/time set 18000 - 将时间设为午夜。

月相

游戏中每过一天，时间计数便会增加24000刻。虽然每天的交替是一样的，但月亮会经历8种月相。虽然没有命令直接更改月相，但/time add 24000命令可以快进至下一个月相。进一步而言，使用以下命令可以直接指定不同的月相：

/time set night - 满月
/time set 38000 - 亏凸月
/time set 62000 - 下弦月
/time set 86000 - 残月
/time set 110000 - 新月
/time set 134000 - 娥眉月
/time set 158000 - 上弦月
/time set 182000 - 盈凸月
