煮熟的肉是能回复最多饱和度的食物。为了能让肉持续吃下去, 就必须养殖一些家畜。 养殖家畜也可以获得一些其他物品:皮革、羽毛、羊毛和鸡蛋。一个手工屠宰农场也可以提供一些经验。

标准的家畜养殖和养殖场

一个标准的农场十分容易建造:
  • 把一些（至少两头/只）动物围起来，使用栅栏围起来是一种经典的方案。
  • 繁殖它们直至达到一个合理的数量。 (或者它们是放养的)。
  • 屠宰它们中的大部分，获得你想要的东西。对剩下的进行繁殖，周而复始……
把周围的动物领回去，你需要拿着可以使他们繁殖的食物来引导他们（如果你有粘液球，拴绳拉回去也不错）：小麦用于牛、哞菇和羊；胡萝卜用于猪和兔子；小麦种子用于鸡。你将他们带入圈中之后，就要开始喂养它们了，这样他们就会进入“求爱模式”，他们会两两分组，诞下幼体。(这样也会产生经验球，注意收集！) 你至快可以每五分钟喂一次动物，从你的成年动物群之中获得幼体。幼体需要大概20分钟才能成长为成年个体，所以别太快太多的杀死成年个体。 在一个确定好的圈内能有多少动物是有自然限制的——通常来讲，如果太挤了，动物就会被推进圈四周的墙内。如果墙体是非透明方块，动物将会窒息；如果墙体是栅栏、玻璃或者其他透明方块，则动物会被挤出去，成为外面的自由生物。一般后者比前者更好一点，因为你可以把逃跑者杀掉。喂养后拥挤导致穿墙这是很普遍的，因为新的幼体可能会把成年个体挤入墙中。 对于羊，一般你不要杀它们，只是把它们染成各种颜色，然后定期剪羊毛就好。他们还需要一个铺满草方块的大羊圈，用于剪毛之后恢复他们的羊毛。 鸡也是比较特殊的，不仅仅因为他们需要用小麦种子来喂养，而且他们还会下蛋，通过扔蛋有几率获得小鸡。这使它们很容易繁殖。需要定期去收集鸡蛋的烦恼可以发展一个蛋场，其中一群鸡保持在水上，它们的蛋将会收集在一个位置。可以用漏斗来实现鸡蛋自动收集，连接到一个投掷器上还能实现自动繁殖。

牛肉生产器



作用
获得牛排，生牛肉和皮革加快速度，变得很简单。

建造
  • 首先放置一个大箱子在前面。装饰一下，在上面放两个物品展示框，在左面一格上面放上生牛肉，在右面一格上面放上牛排。这会提示你按哪个按钮来杀牛。可选项：用一个铁砧来命名箱子为“Meat Locker”。
  • 在箱子之后的2×2区域内放置漏斗，并保证他们都漏向箱子。
  • 在箱子之上的2×2空间放红色染色玻璃。
  • 在漏斗的另一侧放置玻璃后墙。
  • 在漏斗两侧使用不透明方块安置两格宽，3格高的墙（第一层是挨着漏斗的）。作者通常使用砖块来保持视觉上的一致性。
    1. 在最上面的方块的前面，每边放置一个发射器，面朝内侧，在发射器前面放置按钮来控制。放置一桶水在左边的发射器里面，在右边的里面放一桶熔岩。
      2. 在内侧四个实体墙上放置告示牌。这保证你的水或者熔岩不会流下了，只保持在上面一格。 在左侧放置楼梯使得可以走上顶部。
  • 在外部添加一圈楼梯，安装活板门在最上面方块的下部内部腔体的正上方。
准备
  • 用小麦引诱你的牛进入，然后关上活板门，将牛关在里面。然后杀死剩下的牛。这将会是你最后人工杀牛了。
  • 在你的“Meat Locker”里放上一些小麦。
用途
  • 用小麦引诱你的牛进入，然后关上活板门，将牛关在里面。然后杀死剩下的牛。这将会是你最后人工杀牛了。从你的“Meat Locker”拿一些小麦，上楼梯，打开活板门，饲喂你的牛。完了之后关上所有的活板门。
  • 决定一下你是要生牛肉还是牛排（用于食用）。按动对应侧的按钮，等待成年个体死亡，然后再点一下刚刚你按的那个按钮。熔岩会快一些，水可能要几分钟。当然，如果你渴望获得经验，你可以打开活板门进去手动杀牛。
  • 从你的“Meat Locker”里面获得东西然后离开。
如果你需要带一些牛离开去别的地方，打开活板门，点一下放水按钮直到足够的牛离开为止。

猪肉生产器



作用
大幅度提升生猪排与熟猪排的获得速度。含一个自动烤肉机 (existing art). It utilizes a gap made by a redstone repeater with a block above to separate the pigs and piglets.

建造
第一步：烤肉机

1.在你想要造你的机器的那块地方的左边放一个箱子。

2.（从机器的视角）在箱子的前面连接一个漏斗。在漏斗上面放一个“BBQ烧烤炉”（熔炉）。在烧烤炉的右边连接一个漏斗，用来装燃料。在这个漏斗的旁边放一个固体方块，上面弄个告示牌“在这里补充燃料”。

3.在烧烤炉的上方再连接一个漏斗，装满食物（这是“烧烤给料器”）。

第二步：选择器

1.在“烧烤给料器”的后面（箱子的正上方），再连接一个漏斗（这是“收集器”）。接着在箱子顶上，也就是之前那个漏斗的下面连接一个漏斗（这是“选择器”）。

2.在“选择器”的后面放一个固体方块，接着在方块上放一个拉杆。在这个方块的左边 （就像图上那样）再放一个固体方块，上面放一个告示牌，“生的”写在告示牌上面，“熟的”写在告示牌下面。



第三步：分离器

1.在“收集器”的后面连续连接两个漏斗（这是“基层”）。

2. 把“基层”的上方用方块垫住，接着在周围建方块以建造一个2×1×2（2格长，1格宽，2格深）的沟（“屠宰场”）。你更可以在这个沟的右边以玻璃代替方块——这样更便于让你查看猪等生物有没有被淹死。

3. 把这个沟最后面且最低的方块打掉，放个栅栏门，你就造出了一个小型的出入口（边门）。在这个出入口的左边的方块上（什么？没有方块？那就放一个呗）放一个拉杆。在拉杆的旁边再放一个方块，方块上放告示牌，“关“写在告示牌上面，”开“写在告示牌下面。

4.在出入口的后面放一个红石中继器，接着在中继器的上面放一个固体方块。这样这个出入口就只能被小猪通过（只出不进），而成年猪不能。它们是财富……



第四步：准备结束

1.在”屠宰场“的上方建造一个2格深的、封闭的凹陷。 从这个坑往下延伸，一直让它通到”屠宰场“的顶端（这里请参考图）。用墙和屋顶包围来控制猪，把它们带到”屠宰场“的坑里。你可以用玻璃以看见小猪。

2.在凹陷的末尾的顶部放一个朝下发射的发射器，在里面放一桶水。在发射器左边的方块上放一个按钮，在按钮的旁边写上“溺亡装置”。

3.在这个发射器的前面，也就是在顶部方块的后面，放一个活板门。在活板门的前面放一个栅栏门，接着确保用来控制猪的台阶、活板门孔和栅栏门（连接门）衔接得很好，没有让猪逃跑的余地。

4.在发射器的顶上再放一个栅栏门（顶门）。

5. 在连接门的后面造出一片3个方块长的平台（宽度任意），接着从这个平台的右边延伸直到地面的台阶。


准备
把你的猪用胡萝卜引过台阶和栅栏门，最后到达屠宰场。


使用方法
注意：建议把以下这些用简洁的语言标在机器旁边。

1.选择烹饪模式：生的或是熟的。开启边门，关上顶门。这样就可以让猪崽出来（它们不会有掉落物），且不能再进入屠宰场。

2.打开活板门，喂养屠宰场里的猪（或是让它们繁殖），接着关上活板门。仍然拿着萝卜， 走到机器的边缘把小猪引出去。等到所有的小猪都 出去了，把萝卜拿开。

3.出连接门，接着在机器的左边找到“溺亡装置”按钮，按下它。 等到猪死了，再次按下按钮收起水。

4. 关上边门，再次拿起你的萝卜，打开顶门让所有的猪崽进入屠宰场。等到猪崽们进去了，关上活板门，接着从箱子里收集食物。如果你选择了烹饪（熟），那么最好过一会再来收集食物。


修改记录

It is possible to use redstone dust to relocate all controls to the front of the machine or the operator gate area if so desired. It is also possible to use an inverter circuit to ensure the top gate and side gate are always in opposite states.

半自动农场-养羊场

养羊场建造是非常困难的，因为你并不想去杀羊。然而你如果把它们都放在一起剪羊毛又非常不容易。但是如果有一格水或者流动的水，羊会上下跳动，你就可以在一个地方剪羊毛了。接下来就是将这一优势一步步实现的办法。

这个设计可以分分钟剪一百多只羊。

首先你需要建造一个单体水淹农场，但是每一边都要高一点，大约12格。如果你不知道怎么做到这个，请继续往下读或者直接跳到第四步。每层的设计蓝图我已经做好了，在页面的底部可以看到。


第一步：建造一个5×8大小的泥土/草方块平台，就像第三层那样。在草方块的外面，用3层高的方块围起来，防止羊逃跑。
你可以用任意的固体方块（但是不能用栅栏或者玻璃板之类的，羊会卡在里面）但是要留一条给草生长的路径，否则羊毛不会再长出来。

第二步：建造另外一层，同样的尺寸（5×8），用短的一边与上一层相接，使其看起来如同第四层那样。先不要放水，这个要最后办。替代一些方块放置面朝下的活塞（如第五层那样）。这会帮助你控制水流。

第三步：在水体上面放置一层固体方块（在图片中是石头），在上面放上红石中继器，正对着活塞。用红石线连接它们并引到结构的另一端。用一个拉杆来控制开关，要保证所有的活塞在平时是被激活的。如果没有你需要用红石中继器和红石线来修复，确保这一点。当所有的活塞被激活后，你可以放置水在活塞之后，确保水流一层层漫下去，但是在接近或刚好流到坑那个方块时就停止了。就像用水收割的小麦农场一样。

第四步：在图示第三层的最下方有一个2格高和5格宽的缺口，并且在玩家的一边有一排圆石台阶。这就是你的羊会掉进去的坑。在周围建一个3格高的墙，并在最后放置台阶。
台阶让你可以收集羊毛而不用摆放门，并且防止羊跳出养羊场。

第五步：在坑的任意一边会流出水来让羊上下跳动。用固体方块建一个3*4的轮廓，与外围的墙壁相连接。接下来，在水方块和羊平台之间放置2个活塞，确保在蓄水区下面有一个方块。接下来，将红石与活塞连接起来。

第六步：再在水方块的下面一层，把两排朝上的粘性活塞放置于羊坑的位置。然后在上面放置草方块。
粘性活塞不是必要的，普通活塞也行。记得在上面一层建第0层就行了。在活塞一面放置草方块的原因是使羊可以吃更多的草。
用红石中继器指向活塞，并把所有的活塞用红石连接起来。

第七步：把所有的拉杆和红石电路集中到第三层，使用起来更为方便。确保不同的线路之间没有搞错，没有为一些错误的方块充能之后，给各个拉杆贴上标签告示。现在，你应该有一个开关控制第四层的水流（集结开关），一个控制底部两排活塞上下的开关（平台开关），以及最后一个开关，负责控制底下小的水流（收获开关）来让羊上下跳动。测试一下这些功能正常之后，就可以去圈几只羊啦。

警告：当集结开关处于开启状态时，确保收获开关是关闭的／无其他水流。如果两个水流开关同时开启，羊会跳到其他羊上然后逃跑。请记住，如果一个水开关是开着的，那么另外一个必须得是关掉的。详细操作流程如下：

1. 降低平台（平台开关）

2. 释放水流（集结开关）

3. 稍等一会，知道所有的羊都在坑里

4. 关闭集结开关

5. 开启收获开关

6. 拿出剪刀，按住右键，把光标指向羊。在它们上下跳动的时候你会收割许多羊毛。换一换位置来收割以确保覆盖所有的羊。

7. 关掉收获开关和平台开关。平台重新上升。

8. 跳进坑里，一网打尽所有羊毛。

10.收益！
