第四章第八节：添加多材质方块

什么是多材质方块？就是类似草地这样子的方块：绿色的顶部，低端是棕色的，而四周却与上下均不相同。

第一步需要为其创建材质。需要绘制至少两幅材质才可以实现多材质的功能，将其放在同一份子图文件下。

确保使用渲染软件将不同的材质渲染在同一方块上使其看起来彼此拼接，毕竟您不希望一个带有紫色边的方块转过去一看编程绿色边了吧……

在您制作成功材质后实际上代码非常简单，您只需要一个类函数即可：

Magical Multi-texture Method

 
@Override
public int getBlockTextureFromSide(int side)
{
    if (side == 1)
    {
        return 5;
    }
    else if (side == 0)
    {
        return 6;
    }
    else
    {
        return 4;
    }
}

在这一例程中展示了一份草方块的制作过程。通过使用getBlockTexture()类函数，我们可以调用不同位置的子图。

当您看到这一方块时：

顶部：0

底部：1

四周：2、3、4、5

请确认您所看到的方块是否与其吻合~~（据反馈此处有问题……）
