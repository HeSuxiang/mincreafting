一、基础知识：JRE和JDK

1.JRE（Java Runtime Environment Java运行环境）

包括Java虚拟机（JVM Java Virtual Machine）和Java程序所需的核心类库等，运行Java程序只需装JRE即可。

2.JDK（Java Development Kit Java开发工具包）

Java开发工具，包括了JRE，其中的开发工具：编译工具（javac.exe）、打包工具（jar.exe）等。

总的来说就是使用JDK开发java程序，JRE负责运行。

二、JDK下载与安装

1.进入 http://www.oracle.com/technetwork/java/javase/downloads/index.html

2.找到Java SE（建议使用最新版），右边点击JDK下面的DOWNLOAD按钮

3.点击单选框Accept License Agreement

4.32位计算机选windows x86下载，64位计算机选x64下载

5.双击下载下来的文件进行安装，其中安装目录可以更改，不建议安装到C盘，因为重装系统后就会没了。另外其实也可以放到U盘里，也就是说JDK是可以移动的。

如果提示安装Java sdk，可以取消，提示jre也可以不装，因为jdk里面包含了jre。

三、Java环境变量配置

我的电脑（计算机）-右键属性-高级系统设置-环境变量。

1.在系统变量里点新建，变量名为JAVA_HOME，变量值为JDK的安装目录，比如我的是D:\Program Files\Java\jdk1.7.0_09

2.在系统变量里点新建，变量名为Classpath，变量值为.;%JAVA_HOME%\lib;（最前面有个点，最后面有分号，别漏了）

3.在系统变量里找到Path，点编辑，在变量值的最前端加上%JAVA_HOME%\bin;（注意有分号）

4.保存，开始菜单中“运行”，填“cmd”，确定。弹出黑框后输入“javac”，如果有显示java相关信息则表示配置成功。

5.如果有显示java不是内部命令或外部命令则表示配置错误。请仔细检查安装路径和环境变量.

四、配置工作环境

工具下载：MCP： http://mcp.ocean-labs.de/index.php/MCP_Releases

Minecraft Forge-scr： www.minecraftforge.net/forum/index.php/board,3.0.html

Eclipse sdk： http://www.eclipse.org/downloads/

本教程使用jdk1.7.0_09 、Eclipse sdk3.7.0汉化版 、minecraftforge-src-6.4.2.445 、mcp7.23

1.解压mcp.zip到任意目录，比如mcp，建议新建一个mod制作相关的文件夹。

forge安装路径不可以有中文

2.解压forge-src.zip，将里面的forge文件夹放进mcp文件夹里。

3.运行mcp\forge\install.cmd，直到运行完毕（按任意键结束）。

其中第3步会自行下载一些文件，如果下载太慢。可以自行下载下面的文件放进对应目录，再运行install.cmd

Lib.zip-解压并将里面的所有文件放进mcp\lib内

jars.zip-解压并将里面的bin文件夹放进mcp\jars内
