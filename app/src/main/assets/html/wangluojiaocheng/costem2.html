接下来我带来的是手机端附魔教程哦！

1-准备好附魔用品

  1. 你的经验等级(最好大于等于30级)
  2. 你要附魔的东西
  3. 必需的物品→青金石
  4. 书架
  5. 附魔台
  6. 铁砧

2-摆好附魔台和书架

3-现在就可以附魔咯！

(我用钻石剑来代表）

  1. 点附魔台把钻石剑放进去。

  2. 是不是看见三条附魔栏是黑色的。那么就开始放青金石了。

  3. 选择你要附魔那一栏附魔就行了，那么就可以了。当然要消耗经验了～

4-现在是进阶附魔

  1. 放置好铁钻

  2. 准备一本附魔书，我用的是耐久Ⅰ
  3. 放入钻石剑和附魔书，消耗经验完成附魔

  4. 那么你又完成了。
