我的世界命名符彩蛋

此教程由网友水瓶手打制作

又是我，这次我带来的是我的世界彩蛋

彩蛋1-命名符(Dinnerbone/Grumm)

  1. 当然是要铁钻，命名牌和1级的经验
  2. 放好铁钻，打开放入命名牌
  3. 把命名牌消耗一级经验，更改为Dinnerbone/Grumm(第一个字母要大写)
  4. 对准生物使用会让生物上颠倒(玩家和末影龙无效)

彩蛋2-还是命名符(jed_)

  1. 还是铁钻，命名牌和1级的经验
  2. 放好铁钻，打开放入命名牌
  3. 把命名牌消耗一级经验，更改为jeb_
  4. 对准羊使用会让羊变色(只对羊有用，还有用剪刀搜集羊毛掉落的是没有使用命名牌之前的颜色)

彩蛋3-你没猜错，还是命名牌

  1. 还是铁钻，命名牌和1级的经验
  2. 放好铁钻，打开放入命名牌
  3. 把命名牌消耗一级经验，更改为Toast(第一个字母要大写)
  4. 对准兔子使用会让兔子拥有特殊材质(ps:就是加了皮肤)
