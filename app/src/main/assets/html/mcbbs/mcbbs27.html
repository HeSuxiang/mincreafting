常用指令大全

不少新人经常问我某某指令怎么用，某某指令是什么

那今天就教大家各种指令

死亡不掉落

/gamerule keepinventory true

防生物破坏地形

/gamerule mobGriefing false

修改难度

/difficulty 数字（1、简单，2、普通，3、困难）

给予玩家某个物品

/give 玩家名 物品名

杀死某个玩家

/kill 玩家名

修改模式

/gamemode 数字 （0、生存模式，1、创造模式，2、冒险模式）

给予某个玩家所有权限

/op 玩家名

取消某个玩家所有权限

/deop 玩家名

传送到某个玩家身边或者传送到某个坐标

/tp 玩家名或者坐标

（如果玩家名是中文，并且该指令无效的话输入/tp “玩家名”即可）

调整当前的时间

/time set 时间数（0-18000）

关闭火焰蔓延

/gamerule doFireTick false

关闭生物掉落物

/gamerule doMobLoot false

关闭生物生成

/gamerule doMobSpawning false

关闭方块被破坏掉落物品

/gamerule doTileDrops false
