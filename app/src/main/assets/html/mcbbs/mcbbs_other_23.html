jdk安装

那我们就开始准备工作吧！

首先大家电脑里肯定有java，那我就不教大家如何安装java咯！

jdk全称：java Development Kit ，是sun公司针对java开发人员发布的软件开发工具包。

如果要使用编程软件进行开发，需要下载并配置好jdk！

我会在下一章讲解如何配置环境~

http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

点击链接就会看到这么一个页面，可以看到最新的版本是8u131。

我们需要同意用户协议才可以下载，

选择左边的圆圈

【Accept License Agreement】

然后会提示“感谢你同意协议，现在可以下载了。”

选择和你电脑对应的版本下载，

我是windows64位，这里我就选择x64，

如果你的电脑是32位，请选择x86。

mdk安装

mdk全称：Minecraft Development Kit，是mojang提供的MC开发工具包。

这里我提供的是1.7.10版本的已构建mdk，

http://pan.baidu.com/s/1nuZm8AH

如果小伙伴需要其他国内构建好的MDK可以私信我。

其他MDK国外下载：https://files.minecraftforge.net/

下载好jar，打开后就会是如图的样子，

下章会对常用文件夹进行介绍。

Eclipse安装

Eclipse 是一个基于java的开发工具。官方提供了汉化包，请自行下载~

打开链接会看到左边会有这么一个下载，我们点

DOWNLOAD 64BIT

（当然请确保你的电脑是64位的，不是64位的选择下面 Download Packages，选择对应版本）

然后就会跳出“感谢下载eclipse！”

等待跳出下载窗口就行了。

下载好后我们打开这个安装工具

选择第一个安装就好啦~

然后点击

INSTALL

*安装过程较长请耐心等待哟

最后我们就安装好啦~可以打开试试

但你也打不开，因为没有配置环境...
