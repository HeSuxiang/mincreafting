本章目标：

理解并创建项目、包、类，使用Java命名规范命名

什么是项目、包、类？

项目：Project，类似文件夹，存放包、文件夹、使用的系统库。

包：Package，类似文件夹。存放代码用。

类：Class，我们暂时当它是一个文件就好。

ps：一个包跟一个文件夹差不多，但包名有.的话，就像我包名是com.namefind.test，就是com文件夹包含namefind文件夹包含test文件夹。

Java命名规范

大驼峰命名法：首字母大写、第二个词大写。

例：NameFind

小驼峰命名法：首字母小写、第二个词大写。

例：nameFind

项目：没硬性规定，强烈建议使用大驼峰命名法命名

包：以com.开头，然后是自己名字，真·包名

例：com.namefind.entity

*包名必须全部小写

类：使用大驼峰命名法命名

*以上请全部使用英文命名...这是规范~

*请勿使用特殊符号进行命名！

*请勿使用Java关键字，例如使用单一“public”、“class”单词进行命名！

*Java严格区分大小写，请注意！

不懂英文的可以打开翻译软件，顺便学英文233

新建一个普通项目

我们已经导入过一个模组的mdk项目，那么一个普通的java项目是怎样的呢？

右键代码区，New->Java Project

Project name：项目名称

JRE：你的运行环境（理解一下，不用改）

然后点Finish就可以啦。

然后我们就可以得到这么一个普通Java项目，我们来看看它和mdk项目有什么共同的地方。

没错，它们都有一个src文件夹，还有一个JRE System Library（废话-.-）

src：用于存放源代码，你可以理解为主文件夹

JRE System Library：系统的核心类库（只要知道它是系统的东西，我们不去动它就行了...）

新建一个包

我们需要把包放入src文件夹内，所以我们右键src文件夹

New->Package

*mdk项目新建包同理。

Name：你的包名，按照命名规范命名

然后点击Finish~

新建一个类

类分为普通类和主类，区别在于主类里面有一个main方法，普通类里没有。

在这里我们新建两个类、一个普通类、一个主类

右键test包 New->Class

勾选public static void main(String[] args) （main方法）创建为主类，不勾选为普通类。

点击Finish，类就建好了！

ps：它如果生成了一个public 类名( ){ }，就把这段删掉，暂时用不到。
